Flex box com Bootstrap 4

Para criar uma caixa flexível precisamos criar um container com a classe .d-flex. Lembrando que todas as classes aqui descritas em todas as sessões possuem as suas variações responsivas como SM LG XL e MD

Mostrar Código

<div class="d-flex bg-white text-white" style="height:100px;">
<div class="p-2 bg-dark">item 1</div>
<div class="p-2 bg-success">item 2</div>
<div class="p-2 bg-warning"> item 3</div>
<div class="p-2 bg-info"> item 4</div>
<div class="p-2 bg-primary"> item 5</div>
</div>

.justify-content-start | end | center | between | around : Alinhamento dos itens

justify-content

A classe justify-content é usada para alinhar os itens flexíveis, de acordo com cada linha, ou seja, na horizontal, das seguintes maneiras:

  • center: A classe alinha os itens flexíveis no centro do contêiner.
  • flex-start: A classe alinha os items flexíveis no início do contêiner (padrão).
  • flex-end A classe alinha os items flexíveis no final do contêiner.
  • space-around: A classe alinha os itens flexíveis com espaço antes, entre, e depois das linhas.
  • space-between: A classe alinha os itens flexíveis com espaço entre os linhas.

.flex-fill : Itens de preenchimento total e mesmo tamanho

Caso queiramos que os itens preencham toda a largura do container flexível e possuam o mesmo tamanho, precisamos especificar a classe .flex-fill em cada um dos itens que queremos que preencha a largura restante.

align-items-start | end | center | stretch | baseline : Alinhamento dos itens

A propriedade align-items é usada para alinhar os itens flexíveis, de acordo com o tamanho do contêiner, da seguintes maneiras.

  • center: A classe alinha os itens flexíveis no centro do contêiner.
  • flex-start: A classe alinha os items flexíveis na parte superior do contêiner (padrão).
  • flex-end A classe alinha os items flexíveis na parte inferior do contêiner.
  • stretch: A classe estica os itens flexíveis para preencher o contêiner (padrão)
  • baseline: A classe alinha os itens flexíveis de acordo uma linha imaginária na base de todos os conteúdo dos itens. Ou seja, independentemente do conteúdo, nenhum conteúdo ficará abaixo da linha base, mesmo que os contêineres tenham tamanhos distintos.

<div class="d-flex align-items-start flex-wrap bg-white text-white" style="height:150px;" >
<div class="p-2 bg-dark">item 1</div>

<div class="p-2 bg-warning"> item 3</div>

<div class="p-2 bg-primary"> item 5</div>

Order

Podemos alterar a ordem em que os itens são exibidos, sem ter que alterar o código. bastando usar a classe .order-* onde o seu valor varia de 0 a 11.
