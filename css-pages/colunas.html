Múltiplas colunas

  • column-count: propriedade que especifica quantos números de colunas um elemento HTML deve ter.
  • column-gap: Especifica a largura do espaçamento entre as colunas.
  • column-rule-style: Especifica o tipo de divisória utilizada entre as colunas.
  • column-rule-width: Especifica a largura da divisória entre as colunas.
  • column-rule-color: Especifica a cor da divisória entre as colunas.F
  • column-rule: Abreviação
    • column-rule: width style color
  • column-span: Especifica quantas colunas um elemento deve abranger. (imagem ou título por exemplo).
  • column-width: Especifica a largura ideal para cada coluna.
  • columns: Abreviação
    • columns: width count
