list-style: Estilo de Listas

A propriedade list-style é usada para aplicar várias propriedades de estilos em uma lista.

  • list-style-type: Escolhe o simbolo que irá ser usado para identificar cada item de uma lista.
    • Listas Não-Ordenadas: circle (circulo), square (quadrado), disc (disco) e none (sem).
    • Listas Ordenadas: upper-roman (algarismos romanos maiúsculos), lower-roman (algarismos romanos minúsculos), upper-alpha (letras maiúsculas), lower-alpha (letras maiúsculas), none (sem), e OUTROS.
  • list-style-image: Especifica uma imagem como simbolo que irá ser usado para identificar cada item de uma lista.
  • list-style-position: Especifica se os símbolos fazem parte do conteúdo (inside) ou não (outside padrão)
list-style: type position image

Se um dos valores de propriedade acima estiver faltando, o valor padrão para o propriedade ausente será inserida, se houver.

Código Resultado
ul {
list-style-type: upper-roman;
}

Note que, mesmo usando o valor inside para a propriedade list-style-position, ainda há um espaçamento lateral para o objeto que pode ser removido.

Abaixo vamos usar a propriedade list-style, remover os espaçamentos e inserir uma cor de fundo. Lembrando que sempre que usar uma imagem é recomendável usar a propriedade list-style-type, pois assim, caso ocorra algum erro com a imagem, ainda assim, a lista pode estar bem formatada.

Código Resultado
ol {
list: lower-alpha inside url("imagens/listicon.png");
padding: 0;
background-color: black;
}
