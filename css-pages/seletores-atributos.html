Seletores de Atributos

Seletores de atributos são usados para selecionarem elementos de acordo com o tipo ou valor de um determinado atributo.

  • seletor[atributo]{...} : Seleciona todos os elementos especificados que possuem o atributo especificado. Ex.
  • seletor[atributo="valor"] : Seleciona todos os elementos especificados que possuem o atributo especificado exatamente com o valor especificado.
  • seletor[atributo~="valor"] : Seleciona todos os elementos especificados que possuem o atributo especificado que contém em um dos n valores um que seja idêntico ao valor especificado.
  • seletor[atributo|="valor"] : Seleciona todos os elementos especificados que possuem o atributo especificado e contém como primeiro valor o valor especificado. O valor deve ser uma palavra inteira, sozinha. Se contiver duas palavras com hífen (-), ainda assim o valor poder ser considerado como em "valor-texto" e "valor".
  • seletor[atributo^="valor"] : Seleciona todos os elementos especificados que possuem o atributo especificado que contém um valor que começa com o valor especificado. O valor não precisa ser uma palavra inteira, ou seja, considera valores como "valortexto", "valor-texto", "valor" e "valor texto".
  • [attribute$="value"] : Seleciona todos os elementos especificados que possuem o atributo especificado que contém um valor que termina com o valor especificado. O valor não precisa ser uma palavra inteira, ou seja, considera valores como "valortexto", "valor-texto", "valor" e "texto valor".
  • [attribute*="value"] : Seleciona todos os elementos especificados que possuem o atributo especificado que contém um valor que contém o valor especificado. O valor não precisa ser uma palavra inteira, ou seja, considera valores como "valortexto", "textovalor", "valor", "textovalortexto" e "texto-valor" "valor-texto".
