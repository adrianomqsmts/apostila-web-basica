BOM (Browser Object Model) em JavaScript

o BOM permite que o JS converse com o navegador, e por existirem diferentes tipos de navegadores, não existem um padrão definido como o DOM, entretanto os mais modernos implementam quase que da mesma forma as principais métodos e propriedades.

windows : Objeto da janela de navegação!

O objeto window é compatível com todos os navegadores. Representa a janela de um navegador. Todas as variáveis e métodos globais JS são membros de um objeto window assim como oo BOM pode ser acessado pelo window.document.

Janela de Visualização

Podemos mover, fechar, abrir, alterar uma janela de navegação, apensar de que estes métodos não são muito usuais e portanto não serão detalhados.

  • window.open() -Abrir uma nova janela de visualização do navegador
  • window.moveTo() -Mover a posição de uma janela do navegador.
  • window.close() -Mover a posição de uma janela do navegador.
  • resizeTo() -alterar o tamanho de uma janela do navegador.

screen : Informações sobre a janela

O objeto screencontém informações sobre a tela do usuário, como a largura screen.width, a altura screen.height, a largura total screen.availWidth, a altura total screen.availHeight, a quantidade de bits por cores screen.colorDepth, ou a profundidade de pixels de cada tela screen.pixelDepth.

Mostrar Código

<div id="medidas"></div>
<script>
var div = document.getElementById( "medidas" );
var altura = screen.height;
var largura = screen.width;
div.innerHTML = "Altura da tela do usuário " + altura + " - Largura da tela do usuário: " + largura;
</script>


Diferença entre screen.width e screen.availWidth é que um considera a barras de tarefas como o parte da tela e outro não.

Location - Obtendo o endereço da página atual

A propriedade location é usada para obter o endereço atual de uma página WEB (URL) e também redirecionar o navegador para uma nova página.

  • window.location.href - retorna o href (URL) da página atual
  • window.location.hostname - retorna o nome de domínio do host
  • window.location.pathname - retorna o caminho e o nome do arquivo da página atual
  • window.location.protocol - retorna o protocolo da web usado (http: ou https:)
  • window.location.port - retorna o número da porta do host (da página atual)
  • window.location.assign("endereço") - carrega um novo documento.

Mostrar Código

<div id="location"></div>
<script>
var div = document.getElementById( "location" );
var endereco = location.href;
div.innerHTML = "O endereço atual é: " + endereco ;
</script>

history - voltar e avançar

O método history.back() é usado para voltar para a página anterior que o usuário acessou, enquanto o método history.forward() avança a página para uma página que o usuário tinha acessado, mas voltou. Funciona exatamente como os botões avançar e voltar de um navegador, por isso não remos detalhar seu uso.

Caixas pop-up

Caixas pop-up são pequenas janelas que mostram um determinado conteúdo em destaque e de forma independente.

alert : Caixas de alerta

O método alert("mensagem") é usado para alertar o usuário sobre uma ação e é necessário que o usuário clique em "OK" para prosseguir.

Mostrar Código

<button onclick="alert('ALERRRRRTA!!!!')">Clique!</button>

confirm : Caixas de confirmação

O método confirm("mensagem") é usado para enviar uma mensagem ao usuário em que ele tenha que aceitar ao pressionar "OK" ou cancelar pressionando "Cancelar". O método então retorna verdadeiro se "OK" ou falso se "Cancelar"

Mostrar Código

<button id="confirmar" onclick="resposta()">Clique!</button>
<script>
var button = document.getElementById( "confirmar" );
function resposta(){
var resp = confirm("Vamos continuar?");
if (resp){
button.innerHTML = "Você clicou em 'OK'";
} else {
button.innerHTML = "Você clicou em 'CANCELAR'";
}
}
</script>


prompt: Caixas de confirmação e entrada de dados

O método confirm("mensagem", "valorPadrão") é usado para enviar uma mensagem ao usuário em que ele tenha que digitar algum valor, e caso aceite ao pressionar "OK" o valor digitado será retornado pelo método. Caso pressione "Cancelar" o método então irá retornar o valor null. Por isso precisamos tratar o caso o valor seja nulo ou vazio "".

Mostrar Código

<button id="pegarValor" onclick="pegarValor()">Clique!</button>
<script>
var button = document.getElementById( "pegarValor" );
function pegarValor(){
var valor = prompt("Digite o seu nome", "Fulano");
if (valor == null || valor == ""){
button.innerHTML = "Você não digitou um nome!";
} else {
button.innerHTML = "O seu nome é: " + valor;
}
}
</script>

Eventos de Tempo

O objeto window nos permite controlar o tempo que demora uma função ser executada, podendo colocar um atraso para que a função realmente seja executada. Lembre-se que 1 segundo equivale a 1000 milissegundos.

setTimeout() - Executa a função depois de um intervalo de tempo

O método setTimeout(função, milissegundos) chamará a execução da função especificada depois de tantos milissegundos especificados em seu parâmetro. Lembre-se que deve passar apenas o nome da função, e caso a função tenha parâmetro chame desta forma function(){ função(a, b, c); }, no local co parâmetro da função.

Mostrar Código

<button id="tempo-1" onclick="retornoTime = setTimeout(pegarValor,2000)">Clique!</button>
<script>
var button = document.getElementById( "tempo-1" );
function pegarValor(){
var valor = prompt("Digite o seu nome", "Fulano");
if (valor == null || valor == ""){
button.innerHTML = "Você não digitou um nome!";
} else {
button.innerHTML = "O seu nome é: " + valor;
}
}
</script>


setInterval() - Executa a função continuamente com um intervalo de tempo

O método setInterval(função, milissegundos) chamará a execução da função especificada depois de tantos milissegundos especificados em seu parâmetro, depois repetirá chamando a função novamente e novamente após o intervalo de tempo especificado. Lembre-se que deve passar apenas o nome da função, e caso a função tenha parâmetro chame desta forma function(){ função(a, b, c); }, no local co parâmetro da função.

Mostrar Código

<div id="relogio">Clique!</div>
<script>
var div = document.getElementById( "relogio" );
var retornoInterval = setInterval(hora, 1000); function hora(){
var hora = new Date();
hora = hora.toLocaleTimeString();
div.innerHTML = hora;
}
</script>


clearTimeout() e clearInterval() - interrompendo a espera

O método setTimeout() retornam um objeto, e podemos usar este retorno para parar a espera com o método clearTimeout(retorno). Enquanto para parar a espera de um setInterval(retorno) usamos o método clearInterval()

Mostrar Código

<button onclick="clearInterval(retornoInterval)">Parar o Relógio de cima!</button>
