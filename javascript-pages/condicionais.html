Estruturas condicionais em JavaScript

Estruturas condicionais são usadas quando queremos que diferentes ações ocorram em determinados contextos que apresentam diferentes condições.

Estrutura IF - ELSE IF - ELSE

  • if - Usado para executar um determinado trecho de código com base em uma condição se for verdadeira.
  • else if - Usado para executar um determinado trecho de código se a condição if ou if else imediatamente acima não for verdadeira e a condição atual for verdadeira.
  • else- Usado para executar um determinado trecho de código se todas as condições acima não forem verdadeira.

EXEMPLOS:

if(a==b){
//O código entre as chaves vai executar somente se a for igual a b
}


if(a==b){
//O código entre as chaves vai executar somente se a for igual a b
}else {
//O código entre as chaves vai executar somente se a for diferente de b
}


if(a==b){
//O código entre as chaves vai executar somente se a for igual a b
}else if(a > b) {
//O código entre as chaves vai executar somente se a for diferente de b e a maior que b
} else {
//O código entre as chaves vai executar somente se a for diferente de b e menor que b
}


if(a==b){

Estrutura Switch-case

O switch é um estrutura condicional de múltiplos casos, bem semelhante a estrutura IF-ELSE. A estrutura recebe como entrada uma única variável e realiza a comparação igual e de mesmo tipo em cada caso (===), ou seja 10 é diferente de "10".

  • switch(variável) - O switch precisa de um parâmetro único que será usado para realizar a comparação de mesmo valor e tipo em cada caso;
  • case valor: - Cada caso deve ter um valor que será comparado com a variável especificada no switch.
  • break; - É usado para interromper a estrutura, ou seja, quando o break [e executado, as próximas comparações não são executadas.
  • default - Caso especial, colocado como último caso, em que sempre será executado quando nenhum outro caso for executado.

Importante observar o uso do break, pois caso omitido todos os cados abaixo serão executados independentemente se forem verdadeiros ou não. Entretanto podem ocorrer casos em que queremos que mais de um caso seja executado ao mesmo tempo, nesses casos em específicos podemos omitir o uso do break

switch(a){
case 10:
// a é o número 10
break //break impede que o caso "10", 5 e default sejam executados
case "10":
// a é a string "10"
break //break impede que o caso 5 e default sejam executados
case 5:
// a é o número 5
break //break impede que o caso default seja executados
default:
// a é qualquer coisa exceto, 5, 10 e "10"
}

Mostrar Código

<div id="switch"></div>
<script>
var a = 5;
switch(a){
case 10:
var saida = "a é o número 10";
break;
case "10":
var saida = "a é a string \"10\"";
break;
case 5:
var saida = "a é o número 5";
break;
default:
var saida = "a é qualquer coisa exceto, 5, 10 e \"10\"";
}
document.getElementById("switch").innerHTML = saida;
</script>

Conservando o Escopo

Caso queiramos que uma variável criada dentro de uma estrutura de repetição esteja acessível somente dentro da estrutura devemos usar a palavra reservada let ao invés da palavra reservada var.

Exemplo com VAR

Mostrar Código

<div id="escopo"></div>
<script>
var a = 100;
if(a >= 20) {
var a = 20;
}else{
var a = 0;
}
document.getElementById("escopo").innerHTML = a;
</script>



Exemplo com LET

Mostrar Código

<div id="escopo2"></div>
<script>
var a = 100;
if(a >= 20) {
let a = 20;
}else{
let a = 0;
}
document.getElementById("escopo2").innerHTML = a;
</script>
