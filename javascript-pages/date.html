Date em JavaScript

O objeto Date é muito usado para trabalhar com datas e horários. Por ser um objeto pronto, ele já contém propriedades como ano, mês, dia, horas, minutos, segundos e milissegundo que podem ser acessadas após a criação do objeto. Exitem 4 formas de criar um objeto do tipo data.

  • var data = new Date() - Cria um objeto com a data no momento da criação do objeto.
  • var data = new Date(ano, mês, dia, horas, minutos, segundos, milissegundo) - Cria um objeto com a data e horário especificado em valores inteiros.
  • var data = new Date(ano, mês...) - Podemos especificar 5, 4, 3 e 2 valores, onde o primeiro será sempre a data, o segundo o mês, o terceiro o dia, o quarto a hora e assim sucessivamente. Se for especificado apenas um parâmetro, o objeto o interpretará como milissegundos.
  • var data = new Date("texto") - Cria um objeto com a data informada em texto.

Mostrar Código

<div id="data"></div>
<script>
var data =new Date();
document.getElementById("data").innerHTML = data;
</script>


Mostrar Código

<div id="data2"></div>
<script>
var data2 =new Date(1997, 09, 15, 12, 30, 59, 0);
document.getElementById("data2").innerHTML = data2;
</script>


String de Datas

Podemos criar uma nova data especificando ums string como parâmetro, mas devemos seguir uma certa sintaxe. Usaremos o padrão ISO para a criar as datas sendo então: (AAAA-MM-DDTHH:MM:SSZ) onde AAAA [e o ano com 4 dígitos, MM é o mês em dois dígitos (01, 02, 03, etc), DD é o dia em dois dígitos, o caractere T separa a data e a hora, onde HH é a hora em dois dígitos, MM os minutos em dois dígitos e SS os segundos em dois dígitos, finalizado pelo caractere Z.

Podemos omitir a parte do horário, dia e até mesmo mês especificando apenas o ano "AAAA".

Mostrar Código

<div id="data3"></div>
<script>
var data3 =new Date("2019-07-25T12:30:00Z");
document.getElementById("data3").innerHTML = data3;
</script>

Acertando o horário em relação ao UTC

Para adicionar mais horas e minutos em um horário em relação ao UTC precisamos remover o caractere Z e adicionar +HH:MM para incrementar as horas ou -HH:MM para decrementar as horas. ficando algo como (AAAA-MM-DDTHH:MM:SS+HH:MM).

Mostrar Código

<div id="data4"></div>
<script>
var data4 =new Date("2019-07-25T12:30:00+03:30");
document.getElementById("data4").innerHTML = data4;
</script>

toUTCString() : Convertendo datas para o formato UTC

O método data.toUTCString() converte a data para o uma string no formato UTC.

Mostrar Código

<div id="utc"></div>
<script>
var data =new Date();
document.getElementById("utc").innerHTML = data.toUTCString();
</script>

toDateString() : Data mais legível

O método data.toDateString() converte a data padrão em uma string em um formato mais legível.

Mostrar Código

<div id="Date"></div>
<script>
var data =new Date();
document.getElementById("Date").innerHTML = data.toDateString();
</script>

toISOString() : Converte a data para a sintaxe ISO

O método data.toISOString() converte a data padrão para um formato dentro da sintaxe ISO retornando a conversão em uma nova string.

Mostrar Código

get() : Acessando as propriedades das datas

Com o método data.get...() podemos pegar as propriedades de cada data individualmente como o ano, mês e hora.

Método Descrição
getFullYear() Retorna o ano em quatro dígitos numéricos (yyyy)
getMonth() Retorna o mês em um ou dois dígitos numéricos (0-11)
getDate() Retorna o dia em um ou dois dígitos numéricos (1-31)
getHours() Retorna a hora em um ou dois dígitos numéricos (0-23)
getMinutes() Retorna o minuto em um ou dois dígitos numéricos (0-59)
getSeconds() Retorna o segundo em um ou dois dígitos numéricos (0-59)
getMilliseconds() Retorna o milissegundos em um, dois ou três dígitos numéricos (0-999)
getTime() Retorna o tempo em milissegundos desde January 1, 1970
getDay() Retorna a semana em um o dígito numérico (0-6)
getUTCFullYear() Retorna o ano em quatro dígitos numéricos (yyyy)
getUTCMonth() Retorna o mês em um ou dois dígitos numéricos (0-11)
getUTCDate() Retorna o dia no formato UTC
getUTCHours() Retorna a hora no formato UTC
getUTCMinutes() Retorna o minuto no formato UTC
getUTCSeconds() Retorna o segundo no formato UTC
getUTCMilliseconds() Retorna o milissegundos no formato UTC
getUTCDay() Retorna a semana no formato UTC
Date.now() Retorna a data atual .

Mostrar Código

<div id="gets"></div>
<script>
var data =new Date();
document.getElementById("gets").innerHTML = "Ano: " + data.getFullYear() + "<br>" +
"Mês: " + data.getMonth() + "<br>" +
"Dia: " + data.getDate() + "<br>" +
"Hora: " + data.getHours() + "<br>" +
"minutos: " + data.getUTCMinutes() + "<br>" +
"segundos: " + data.getSeconds() + "<br>" +
"Milissegundos: " + data.getMilliseconds() + "<br> <br>" +
"Milissegundos desde 1 de Janeiro de 1970: " + data.getTime() + "<br>" +
"Dia da semana: " + data.getDay() + "<br>" +
"Data Atual: " + Date.now();
</script>


Como o mês e o dia da semana estão em formato numérico precisamos criar um array com o nome de cada mês (janeiro, fevereiro, ...) ou cada dia da semana (domingo, segunda, ...) e usar o acesso de vetor na posição de mesmo valor numérico para acessar o valor em texto do array.

Mostrar Código

<div id="semana"></div>
<script>
var data =new Date();
var semana = ["Domingo","Segunda-feira", "Terça-feira", "Quarta-feira", "Quinta-feira", "Sexta-Feira", "Sábado"];
document.getElementById("semana").innerHTML = "Dia da Semana: " + semana[data.getDay()];
</script>

set() : Alterando os calores das datas

Os métodos data.set..(valor) são usados para alterar os valores de cada propriedade de um objeto do tipo data antes já declarado de forma individual sem que tenhamos que criar um novo objeto.

Método Descrição
setDate() Altera o dia com um valor numérico (1-31)
setFullYear() Altera o ano (Pode opcionalmente alterar adicionando o mês e o dia yyyy-mm-dd)
setHours() Altera a hora com um valor numérico (0-23)
setMilliseconds() Altera os milissegundos com um valor numérico (0-999)
setMinutes() Altera os minutos com um valor numérico (0-59)
setMonth() Altera o mês com um valor numérico (0-11)
setSeconds() Altera os segundos com um valor numérico (0-59)
setTime() Altera o milissegundos desde 1º de janeiro de 1970, basicamente muda a data.

Mostrar Código

<div id="set"></div>
<script>
var data =new Date();
data.setHours(18);;
document.getElementById("set").innerHTML = "Hora: " + data.getHours();
</script>
