DOM (Document Object Model) em JavaScript

O DOM é usado para acessar e alterar todos os elementos de um documento HTML, como alterar os elementos, atributos, estilos, remover elementos, adicionar elementos, reagir a eventos e criar novos eventos, tudo isso com JS.

DOM é um padrão de acesso aos documentos criada pelo W3C (World Wide Web Consortium), enquanto o HTML DOM é um modelo de objeto e de interface de programação HTML de como obter, alterar, adicionar ou excluir elementos em HTML.

Encontrando Elementos HTML

Existem diversas formas de encontrar um elemento HTML dentro de um documento, sendo os mais clássicos por identificador e por classes.

  • document.getElementById("identificador"); - É usado para capturar o elemento HTML pelo identificador.
  • document.getElementsByTagName("p"); - É usado para capturar todos os elementos HTML que são da tag especificada. Para ver detalhes sobre os elementos retornados acesse:HTMLCollection
  • document.getElementsByClassName("classe"); - É usado para capturar todos os elementos HTML que possuem a classe especificada.
  • document.querySelectorAll("p.classe"); - É usado para capturar todos os elementos HTML que a classe especificada e que sejam do elemento especificado, igualmente como funcionam os seletores de CSS Seletores I.
  • document.forms["id"]; - É usado para capturar todos os elementos HTML do formulário especificado.
  • document.body - Captura todos os elementos dentro do corpo do documento.
  • document.documentElement - Captura todos os elementos html dentro do documento.
  • document.embeds - Captura todos os elementos <embed> no documento.
  • document.forms - Captura todos os elementos <form> no documento.
  • document.head - Captura todos os elementos dentro do elemento <head> do documento.
  • document.images - Captura todos os elementos <img> no documento.
  • document.links - Captura todos os elementos <a> no documento.
  • document.scripts - Captura todos os elementos <scripts> do documento.
  • document.title - Captura todos os elementos <title> do documento.


Mostrar Código

<div id="identificador"></div>
<script>
document.getElementById("identificador").innerHTML = " Alterando o valor pelo identificador";
</script>


No caso de seleção de vários elementos HTML, teremos um vetor e devemos trabalhar sobre ele.

Alterando o HTML

Como já vimos até aqui, podemos usar a propriedade innerHTML para alterar o conteúdo dos elementos HTML, inclusive adicionando novos elementos dentro deles.

Aletrando o atributo de um HTML

Para alterar um atributo de um elemento HTML precisamos especificar o nome do atributo e seu novo valor .atributo = . Entre os alguns exemplos de atributos que podemos mudar são: class, src, id, href, etc...

Mostrar Código

<div id="cor">Elemento</div>
<script>
var div = document.getElementById("cor");
div.className = "bg-warning";
</script>

DOM CSS - alterando o estilo dos elementos HTML

Para alterar todo o estilo de um elemento HTML usamos a propriedade .style = "color: red, ..." ou alteramos individualmente cada propriedade de estilo como em .style.color = "red".

<div id="css">Elemento</div>

var div = document.getElementById("css");

Eventos

Eventos são basicamente uma interação com algum elemento HTML como clicar sobre um elemento, carregamento da página ou alteração sobre um campo de entrada. E podemos usar o JS para fazer uma reação com a interação. A propriedade .this e usada para especificar o elemento atual.

Evento Descrição
onchange Um elemento HTML foi alterado/carregado
onclick Um clique de um usuário sobre um elemento
onmouseover O usuário move o mouse sobre um elemento HTML
onmouseout O usuário move o mouse para longe de um elemento HTML
onkeydown O usuário pressiona uma tecla do teclado
onload O navegador terminou de carregar a página

Esses eventos são atributos de elementos HTML e através deles podemos chamar um função ou código JS para realizar uma reação.

Mostrar Código

<button onclick="window.alert('ALERRRRRTA!!!!')">Clique!</button>

Mostrar Código

<div id="funcao2" onmouseover="exibirData()" onmouseout="this.innerHTML = `Passe o Mouse!`" class="p-5 d-inline-block bg-warning">Passe o Mouse!</div>
<script>
function exibirData() {
document.getElementById("funcao2").innerHTML = Date();
}
</script>
Passe o Mouse!

Adicionando eventos com JS

Podemos também adicionar um elemento via JS através do método addEventListener("nomeEvento", nomeFunção). Com isso você pode adicionar mais de um evento do mesmo tipo, como dois eventos de clicar, sem sobrescrever o anterior, além de ajudar a limpar mais o código HTML deixando-o menos poluído. E para remover um evento basta usar o método removeEventListener("nomeEvento", nomeFunção).

Observe que para usar o nome de evento não precisa usar o prefixo "on", ficando apenas "click" ao invés de "onclick". E na chamada de uma função devemos apenas passar o seu nome (sem os parênteses), em caso de funções com parâmetros devemos usar chamar function() { minhafuncao(parametro1, parametro2);}

Mostrar Código

<div id="addEvento">CLIQUE AQUI!</div>
<script>
function alerta() {
alert("Hello World!");
}
function colorir(cor) {
var e = document.getElementById("addEvento");
e.style.background = cor;
}
var elemento = document.getElementById("addEvento");
elemento.addEventListener("click", alerta );
elemento.addEventListener("click", function(){ colorir( "blue" ); } );
</script>
CLIQUE AQUI!

Ordem dos eventos

Para alterar a ordem em que ocorrem os eventos em elementos aninhados com eventos, devemos usar o terceiro parâmetro da função addEventListener("NomeEvento", nomeFunção, true/false);, onde se o valor do terceiro parâmetro for true então o elemento mais externo irá ativar o seu evento e somente então o mais interno ativará o evento. Por padrão o terceiro parâmetro é false, assim o evento interno será acionado primeiro e somente os eventos dos elementos mais externos poderão ser acionados.

Navegando na árvore da família de Elementos

Pela imagem podemos ter um noção do parentesco entre os elementos HTML onde o elemento <html> não possui pais, mas possui dois filhos, o elemento <head>e o elemento <body>, e ambos são elementos irmãos. O elemento <body> tem o pai <html> e tem os filhos <p> e <div>, e assim sucessivamente.

Para acessarmos os elementos HTML pelo parêntescos devemos usar as propriedades:

  • parentNode - Acessa o pai de um elemento HTML
  • childNodes[numero] - Acessa o filho da posição especificada de um elemento HTML
  • firstChild - Acessa o primeiro filho de um elemento HTML
  • lastChild - Acessa o último filho de um elemento HTML
  • nextSibling - Acessa o próximo irmão de um elemento HTML
  • previousSibling - Acessa o irmão anterior de um elemento HTML

Mostrar Código

<div id="pai">
<span>Elemento Span</span>
</div>
<script>
var filho = document.getElementById("pai").firstChild;
filho.style.color = "black";
filho.style.backgroundColor = "white";
</script>
Elemento Span

nodeName: Nome de um nó

A propriedade nodeName é usada para pegar o nome do elemento HTML como div, p, span, a, etc. Um nodeName não pode ser alterado, e pode usado para ver o nome de um atributo também. Para ver detalhes sobre os nós acesseNode Lists

Mostrar Código

<div id="nodeName"></div>
<script>
var div = document.getElementById("nodeName");
var tag = document.getElementById("nodeName").nodeName;
div.innerHTML = tag;
</script>



nodeValue : O valor de um nó

A propriedade nodeValue retorna o valor de um elemento HTML. Olhando na imagem acima, você irá notar que alguns elemento tem atributos e classes e eles são FILHOS de um elemento HTML. Ou seja, para acessar o valor de um conteúdo de um elemento HTML devemos acessar o seu filho! (Sim, é o mesmo que usar o innerHTML). Se for usado para retornar o valor de um elemento HTML o valor será null, enquanto de um texto será o próprio texto e o atributo será o valor do atributo.

Mostrar Código

<div id="nodeValue">Texto é um filho do elemento! </div>
<div id="outraDiv"></div>
<script>
var div = document.getElementById("nodeValue");
var outraDiv = document.getElementById("outraDiv");
var txt = document.getElementById("nodeValue").firstChild.nodeValue;
outraDiv.innerHTML = txt;
</script>
Texto é um filho do elemento!



nodeType : Tipo do nó

A propriedade noType retorna o tipo do nó que estamos trabalhando, podendo ser ELEMENT_NODE ( 1- ELEMENTO html), TEXT_NODE (3 - texto), COMMENT_NODE (4 - comentário), etc.

Mostrar Código

<div id="nodeType"></div>
<script>
var div = document.getElementById("nodeType");
var tipo = div.nodeType;
div.innerHTML = tipo;
</script>

Criando elementos HTML

Para criar um elemento HTML primeiro precisamos criar um nó através do método noElemento = document.createElement("elementoHTML");, depois podemos criar o conteúdo para o nó através do método noTexto = document.createTextNode("texto");. Próximo passo unir o nó do elemento como pai do nó texto (filho) através do método noElemento.appendChild(noTexto);.

Por fim precismos anexar o nó elemento criado dentro do nosso documento, dentro de outro elemento. para isso devemos apenas chamar novamente o método pai.appendChild(noElemento); para anexar o novo nó em algum elemento existente.

Mostrar Código

<div id="novoElemento"></div>
<script>
var noElemento = document.createElement("span");
var noTexto = document.createTextNode("Span criado");
noElemento.appendChild(noTexto);
var pai = document.getElementById("novoElemento");
pai.appendChild(noElemento);
</script>

Ainda podemos criar atributos (document.createAttribute) para um nó, criar comentários (document.createComment), criar eventos (document.createEvent), dentre outras coisas.

Inserindo em qualquer posição

O método appendChild insere o novo nó sempre oa final do elemento pai, para inserir no inicio precisamos usar o método pai.insertBefore(nóElemento, elementoIrmão). Com ese método o novo nó será inserido na posição atual do elemento irmão.

Mostrar Código

<div id="posicao">
<span id="i1">irmão 1</span>
<span id="i2">irmão 2</span>
<span id="i3">irmão 3</span>
</div>
<script>
var noElemento = document.createElement("span");
var noTexto = document.createTextNode("NOVO IRMAO ");
noElemento.appendChild(noTexto);
var pai = document.getElementById("posicao");
var irmao = document.getElementById("i2");
pai.insertBefore(noElemento, irmao);
</script>
irmão 1 irmão 2 irmão 3

Removendo um elemento HTML

Para remover um elemento HTML basta usar o método elemento.remove().

Mostrar Código

<div id="remover">EU FUI REMOVIDO! </div>
<script>
var remover = document.getElementById("remover");
remover.remove();
</script>
EU FUI REMOVIDO!

Substituindo um elemento HTML

Para substituir um elemento HTML basta usar o método pai.replaceChild(noElemento, irmãoRemover).

Mostrar Código

<div id="substituir">
<span id="i-1">irmão 1</span>
<span id="i-2">irmão 2</span>
<span id="i-3">irmão 3</span>
</div>
<script>
var noElemento = document.createElement("span");
var noTexto = document.createTextNode("NOVO IRMAO ");
noElemento.appendChild(noTexto);
var pai = document.getElementById("substituir");
var irmao = document.getElementById("i-2");
pai.replaceChild(noElemento, irmao);
</script>
irmão 1 irmão 2 irmão 3
