Laços de repetição em JavaScript

Um estrutura de repetição basicamente é usada para repetir um mesmo trecho de código um número finito de vezes, tendo como objetivo reduzir o número de linhas escritas com o mesmo trecho de código.

SEMPRE ESPECIFIQUE A CONDIÇÃO DE PARADA!!! Caso esqueça ou erre a lógica das condições o laço de repetição nunca irá parar e poderá travar o sistema.

Estrutura de repetição FOR

O for(condição1; condição2; condição3) é usado quando queremos repetir uma QUANTIDADE CONHECIDA de vezes um determinado trecho de código.

  • Condição 1 - É executada uma única vez, geralmente é usada para inicializar o valor de um contador com o valor 0; CONHECIDA COMO CONDIÇÃO INICIAL
  • Condição 2 - É uma condição que sempre será executada no inicio cada interação e normalmente define um valor limite de quantas vezes queremos que a estrutura seja executada. CONHECIDA COMO CONDIÇÃO DE PARADA
  • Condição 3 - É executada sempre ao final da execução do bloco de código, geralmente é usada para incrementar o valor do contador; CONHECIDA COMO CONDIÇÃO DE INCRIMENTO

As condições podem ser omitidas e a estrutura pode assumir diversos formatos que normalmente podem levar a uma má legibilidade dos códigos e não são muito tradicionais, e por estes motivos não iremos abordar essas possibilidades. Um caso mais comum é usar um segunda condição de parada dentro do código com um if e um break.

for (var i = 0; i < 5; i++){
// Trecho de código que irá ser executado 5 vezes, pois o valor de i será incrementado uma unidade a cada execução (como também poderia incrementado de 5 em 5, 5 em 5 e não necessariamente 1 em 1), e tem a condição de parada para i menor que 5. Lembrando que i começa em 0.
}

Mostrar Código

<div id="for1"></div>
<script>
var texto = "";
for (var i = 0; i < 5; i++) {
texto += "O contador i agora é : " + i + "<br>";
}
document.getElementById("for1").innerHTML = texto;
</script>


A condição 1 é opcional, pois podemos declarar o seu valor anteriormente, a condição de parada pode ser substituída por uma estrutura IF break para interromper a estrutura FOR (dentro do FOR) e a condição de incremento pode ser omitida e mais uma vez pode ser feita dentro da estrutura FOR. Como dito anteriormente essas outras possibilidades devem ser evitadas.

Mostrar Código

<div id="for2"></div>
<script>
var texto = "";
var i = 0;
for (; ;) {
texto += "O contador i agora é : " + i + "<br>";;
i++;
if(i >= 5) {
break;
}
}
document.getElementById("for2").innerHTML = texto;
</script>

Estrutura de Repetição FOR-IN

A estrutura for(variável in objeto) segue a mesma lógica de repetição quem um for tradicional, entretanto é usado para percorrer todos as propriedades objeto retornando um valor inteiro da posição da propriedade e NÃO o valor da propriedade.

for (i in objeto){
// Trecho de código que irá ser executado na mesma quantidade de propriedades que tem dentro do objeto especificado. Onde i é um valor inteiro para acessar o vetor de propriedades de um objeto.
}

Mostrar Código

<div id="for-in"></div>
<script>
var pessoa = {
nome: "Fulano Ciclano de Tal",
idade: 47,
altura: 1.68,
}
var texto = "";
for (i in pessoa) {
texto += "valor: " + pessoa[i] + "<br>";
}
document.getElementById("for-in").innerHTML = texto;
</script>

Estrutura de repetição FOR-OF

A estrutura de repetição for(variável of objeto ) é usada para PERCORRER OS VALORES de um objeto, ou seja, o contador conterá o valor de cada propriedade do objeto. Também pode ser usado para percorrer os valores de um array ou os caracteres de uma string.

Mostrar Código

<div id="for-of1"></div>
<script>
var array = ["Cereja", "Abacate", "Morango", "Manga", "Goiaba", "Amora"];
var texto = "Frutas: ";
for (i of array) {
texto += i + " ";
}
document.getElementById("for-of1").innerHTML = texto;
</script>


Mostrar Código

<div id="for-of2"></div>
<script>
var palavra = "Letras";
var texto = " - ";
for (i of palavra) {
texto += i + " - ";
}
document.getElementById("for-of2").innerHTML = texto;
</script>

Estrutura de repetição WHILE

A estrutura de repetição while(condição){ CÓDIGO } é semelhante à estrutura for sem a condição 1 e condição 3. Ou seja, devemos em muitos casos iniciar um valor anteriormente e criar uma condição de incremento dentro do corpo da estrutura. Os principais usos de uma estrutura while estão relacionados quando não sabemos quantas vezes iremos executar o bloco de código. Seu uso mais comum contém a palavra reservada break.

Mostrar Código

<div id="while1"></div>
<script>
var texto = "";
var i = 0;
while (i <= 5) {
texto += "O contador i agora é : " + i + "<br>";
i++;
}
document.getElementById("while1").innerHTML = texto;
</script>

Estrutura de repetição DO WHILE

A estrutura de repetição do{ }while(condição) é semelhante à estrutura while. A única diferença que a condição é verificada ao final da execução do código e não no início. Ou seja, o do while executará sempre um única vez independentemente se a condição for verdadeira ou não. Enquanto o while sempre fará a verificação da condição antes de executar o código.

Mostrar Código

<div id="dowhile"></div>
<script>
var texto = "";
var i = 0;
do {
texto += "O contador i agora é : " + i + "<br>";
i++;
} while (i <= 5)
document.getElementById("dowhile").innerHTML = texto;
</script>


Mostrar Código

<div id="dowhile1"></div>
<script>
var texto = "";
var i = 7;
do {
texto += "O contador i agora é : " + i + "<br>";
i++;
} while (i <= 5)
document.getElementById("dowhile1").innerHTML = texto;
</script>

Conservando o Escopo

Caso queiramos que uma variável usada dentro do escopo de ums estrutura de repetição seja visível somente dentro do bloco, então devemos usar a palavra reservada let ao invés da palavra reservada var

Exemplo com VAR

Mostrar Código

<div id="var"></div>
<script>
var texto = "";
var i = 100;
for (var i = 0; i < 5; i++) {
texto += "O contador i agora é : " + i + "<br>";
}
document.getElementById("var").innerHTML = texto + "<br>" + "valor do contator i ao final é : " + i;
</script>


Exemplo com LET

Mostrar Código

<div id="let"></div>
<script>
var texto = "";
var i = 100;
for (let i = 0; i < 5; i++) {
texto += "O contador i agora é : " + i + "<br>";
}
document.getElementById("let").innerHTML = texto + "<br>" + "valor do contator i ao final é : " + i;
</script>
