Efeitos

Podemos aplicar vários efeitos nos elementos, dado um evento, com os métodos JQuery. Uma losta, com exemplo e descrição dos métodos podem ser acessadas em w3schools jQuery Effect Methods

Ocultar e Mostrar

O método hide() oculta o elemento, o método show() desoculta um elemento, enquanto o método toggle() oculta elementos mostrados e mostra elementos ocultados. Os três métodos podem receber dois parâmetros opcionais, o primeiro especificando a velocidade em que o elemento será ocultado/mostrado, podendo ser "slow", "fast" ou um tempo específico em milissegundos. O segundo parâmetro recebe uma função de callback


$("#btToggle").click(function () {
  $("#toggleEX").toggle();
});

Aparecimento e Desaparecimento

O método fadeIn() faz um elemento oculto aparecer, o método fadeOut() faz um elemento visível desaparecer, enquanto o método fadeToggle faz um elemento oculto aparecer e um elemento visível desaparecer. Os três métodos podem receber dois parâmetros opcionais, o primeiro especificando a velocidade em que o elemento será ocultado/mostrado, podendo ser "slow", "fast" ou um tempo específico em milissegundos. O segundo parâmetro recebe uma função de callback.

$("#btnfade").click(function () {
  $("#divfade").fadeToggle();
});

Slide

O método slideDown() faz um elemento oculto aparecer, o método SlideOut() faz um elemento visível desaparecer, enquanto o método slideToggle faz um elemento oculto aparecer e um elemento visível desaparecer. Os três métodos podem receber dois parâmetros opcionais, o primeiro especificando a velocidade em que o elemento será ocultado/mostrado, podendo ser "slow", "fast" ou um tempo específico em milissegundos. O segundo parâmetro recebe uma função de callback.

$("#btnslide").click(function () {
  $("#divslide").slideToggle();
});

Animação

O método animate permite criar animações customizáveis, onde recebe uma lista de parâmetros de atributos CSS entre chaves {}, além de especificar a velocidade da animação como "slow", "fast" ou um tempo específico em milissegundos, além de um terceiro parâmetro uma função de callback.

$("#btnanimate").click(
  function () {
    $("#divanimate").animate({
      left: '250px',
      opacity: '0.5',
      fontSize: '20px',
      height: '150px',
      width: '150px'
    }, 5000);
  }
);

Para usar valores relativos, onde o valor é então relativo ao valor atual do elemento). Isso é feito colocando += ou -= na frente do valor.

Para Criar uma fila de animações, basta chamar o método animate várias vezes em seguida, e então elas irão executar em sequência.

        $("#btnanimate").click(
          function () {
            $("#divanimate").animate({
              left: '250px',
              opacity: '0.5',
              fontSize: '20px',
              height: '150px',
              width: '150px'
            }, 5000);

            $("#divanimate").animate({
              left: '500px',
              opacity: '1',
              fontSize: '14px',
              height: '200px',
              width: '200px'
            }, 5000);
          }
        );

stop

O método stop() é usado para interromper uma animação, slide, etc.

$("#stop").click(function () {
  $("#panel").stop();
});

Usando Callback

Uma função de retorno de chamada é executada após o efeito atual estar 100% concluído.

$("button").click(function(){
  $("p").hide("slow", function(){
    alert("The paragraph is now hidden");
  });
});
