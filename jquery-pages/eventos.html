Eventos

Um evento representa o momento preciso em que algo acontece. O termo "dispara / dispara" é freqüentemente usado com eventos. Exemplo: "O evento de pressionamento de tecla é disparado no momento em que você pressiona uma tecla"

Sintaxe

A maioria dos eventos DOM tem um equivalente em JQuery. Para construir um evento, basta chamar o método após a seleção do elemento. Após chamar o evento, devemos definir uma função que irá ocorrer após o disparo do evento.

$(elemento).evento(function(){...});

Lista de Eventos

Uma lista completo, com exemplos e descrição dos eventos em JQuery estão disponíveis em w3schools jQuery Event Methods


ready()

O evento ready nos permite executar uma função quando o documento está totalmente carregado.

$(document).ready(function () {
  $("button").click(function () {
    $(".test").hide();
  });
});

click()

O método executa a função quando o usuário clicar sobre o elemento selecionado

  $("button").click(function () {
    $(".test").hide();
  });

dbclick()

O método executa a função quando o usuário der dois cliques sobre o elemento selecionado.

  $("button").dbclick(function () {
    $(".test").hide();
  });

mouseenter()

O método executa a função quando o ponteiro do mouse entrar sobre o elemento selecionado

  $("button").mouseenter(function () {
    $(".test").hide();
  });

mouseleave()

O método executa a função quando o ponteiro do mouse sair da área sobre o elemento selecionado

  $("button").mouseleave(function () {
    $(".test").hide();
  });

mousedown()

O método executa a função quando com o botão esquerdo, direito ou central do mouse estiver pressionado sobre o elemento selecionado.

  $("button").mousedown(function () {
    $(".test").hide();
  });

mouseup()

O método executa a função quando com o botão esquerdo, direito ou central do mouse estiver sendo despresssionado sobre o elemento selecionado.

  $("button").mouseup(function () {
    $(".test").hide();
  });

hover()

O método executa duas funções, a primeira quando o mouse passa sobre o elemento selecionado, e a segunda quando o mouse saí da área do elemento selecionado.

$("#p1").hover(
  function(){
    alert("You entered p1!");
  },
  function(){
    alert("Bye! You now leave p1!");  
  }
);

focus()

O método executa a função quando um campo de formulário recebe o foco

$("input").focus(function(){
  $(this).css("background-color", "#cccccc");
});

blur()

O método executa a função quando um campo de formulário deixa de estar focado

$("input").blur(function(){
  $(this).css("background-color", "#fffff");
});

on()

O método anexa um ou mais eventos para os elementos selecionados, caso algum destes eventos ocorram a função será executada.

$("p").on("click", function(){
  $(this).hide();
}); 

Múltiplos eventos

$("p").on({
  mouseenter: function(){
    $(this).css("background-color", "lightgray");
  },
  mouseleave: function(){
    $(this).css("background-color", "lightblue");
  },
  click: function(){
    $(this).css("background-color", "yellow");
  }
});
