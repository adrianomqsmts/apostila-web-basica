Filtro de Elementos

  • first() - O método retorna o primeiro elemento dos elementos especificados
  • last() - O método retorna o último elemento dos elementos especificados.
  • eq() - O método retorna um elemento com um número de índice específico dos elementos selecionados. Os números do índice começam em 0, então o primeiro elemento terá o número do índice 0 e não 1.
  • filter() - O método permite que você especifique um critério. Os elementos que não correspondem aos critérios são removidos da seleção e aqueles que correspondem serão retornados.
  • not() - O método retorna todos os elementos que não correspondem aos critérios.
$(document).ready(function () {
  /* O exemplo a seguir seleciona o primeiro <div> elemento */
  $("div").first();
  /* O exemplo a seguir seleciona o último <div> elemento:*/
  $("div").last();
  /*O exemplo a seguir seleciona o segundo <p>elemento (número do índice 1):*/
  $("p").eq(1);
  /*O exemplo a seguir retorna todos os <p>elementos com o nome de classe "intro":*/
  $("p").filter(".intro");
  /* O exemplo a seguir retorna todos os <p>elementos que não têm o nome de classe "intro":*/
  $("p").not(".intro");
});
