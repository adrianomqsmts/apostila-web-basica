Manipular o HTML

O jQuery vem com vários métodos relacionados ao DOM que facilitam o acesso e a manipulação de elementos e atributos.

Obter Conteúdo

  • O método text()define ou retorna o conteúdo de texto dos elementos selecionados
  • O método html() define ou retorna o conteúdo dos elementos selecionados (incluindo marcação HTML)
  • O método val() define ou retorna o valor dos campos do formulário
  • O método attr() define ou retorna os valores dos atributos dos elementos selecionados.
$(document).ready(function () {
  $("#btn1").click(function () {
    alert("Text: " + $("#test").text());
  });
  $("#btn2").click(function () {
    alert("HTML: " + $("#test").html());
  });
});

Alterar Conteúdo

Os métodos abaixo também possuem uma função de retorno, que recebe dois parâmetros. O primeiro informa o índice do elemento atual (tem uma lista de elementos selecionados), e o segundo parâmetro informa o valor original que estava no elemento.

$(document).ready(function(){
  $("#btn1").click(function(){
    $("#test1").text(function(i, origText){
      return "Old text: " + origText + " New text: Hello world! (index: " + i + ")"; 
    });
  });
});

});

Adicionar Conteúdo

Veremos quatro métodos jQuery que são usados ​​para adicionar novo conteúdo:

  • append() - Insere conteúdo no final dos elementos selecionados
  • prepend() - Insere conteúdo no início dos elementos selecionados
  • after() - Insere conteúdo após os elementos selecionados
  • before() - Insere conteúdo antes dos elementos selecionados
$("#divex").append("append");
$("#divex").prepend("prepend");
$("#divex").after("after");
$("#divex").before("before");

Remover Conteúdo

Para remover elementos e conteúdo, existem principalmente dois métodos jQuery:

  • remove()- Remove o elemento selecionado (e seus elementos filho)
    • O método também aceita um parâmetro, que permite filtrar os elementos a serem removidos.
  • empty()- Remove os elementos filhos do elemento selecionado

Este exemplo remove todos os P elementos com class="test"e class="demo"

$("#div1").empty();
$("#div1").remove();
$("p").remove(".test");
$("p").remove(".test, .demo");
