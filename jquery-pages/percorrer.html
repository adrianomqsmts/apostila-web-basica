Percorrer Elementos

Pela imagem podemos ter um noção do parentesco entre os elementos HTML onde o elemento <html> não possui pais, mas possui dois filhos, o elemento <head>e o elemento <body>, e ambos são elementos irmãos. O elemento <body> tem o pai <html> e tem os filhos <p> e <div>, e assim sucessivamente.

Percorrer Ancestrais

  • parent() - O método retorna o elemento pai direto do elemento selecionado. Este método atravessa apenas um único nível na árvore DOM.
  • parents() - O método retorna todos os elementos ancestrais do elemento selecionado, até o elemento raiz do documento
    • Você também pode usar um parâmetro opcional para filtrar a pesquisa de ancestrais.
  • parentsUntil() - O método retorna todos os elementos ancestrais entre dois argumentos fornecidos

O exemplo a seguir retorna o elemento pai direto de cada elemento SPAN

$("span").parent();


O exemplo a seguir retorna todos os ancestrais de todos os elementos SPAN

$("span").parents();


O exemplo a seguir retorna todos os ancestrais de todos os elementos SPAN que são elementos UL

$("span").parents("ul");


O exemplo a seguir retorna todos os elementos ancestrais entre um e SPAN um elemento DIV

$("span").parentsUntil("div");


Percorrer Descendentes

  • children() - O método retorna todos os filhos diretos do elemento selecionado. Este método percorre apenas um único nível abaixo da árvore DOM.
    • Você também pode usar um parâmetro opcional para filtrar a pesquisa de filhos.
  • find() - O método retorna elementos descendentes do elemento selecionado, até o último descendente.
    • Você também pode usar um parâmetro opcional para filtrar a pesquisa de filhos.

O exemplo a seguir retorna todos os elementos que são filhos diretos de cada elemento DIV

$("div").children();


O exemplo a seguir retorna todos os elementos P com o nome de classe "primeiro", que são filhos diretos de DIV

$("div").children("p.first");


O exemplo a seguir retorna todos os elementos SPAN que são descendentes de DIV

$("div").find("span");


O exemplo a seguir retorna todos os descendentes de DIV

$("div").find("*");


Percorrer os Irmãos

Existem muitos métodos jQuery úteis para percorrer lateralmente na árvore DOM. Você também pode usar um parâmetro opcional para filtrar a pesquisa por irmãos

  • siblings() - O método retorna todos os elementos irmãos do elemento selecionado
  • next() - O método retorna o próximo elemento irmão do elemento selecionado
  • nextAll() - O método retorna todos os próximos elementos irmãos do elemento selecionado.
  • nextUntil() - O método retorna todos os próximos elementos irmãos entre dois argumentos fornecidos

Os métodos prev(), prevAll() e prevUntil() funcionam exatamente como os métodos acima, mas com funcionalidade reversa: eles retornam elementos irmãos anteriores (retrocedem ao longo dos elementos irmãos na árvore DOM, em vez de avançar).
