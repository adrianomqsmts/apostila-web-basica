Seletores

Os seletores jQuery são usados ​​para "encontrar" (ou selecionar) elementos HTML com base em seu nome, id, classes, tipos, atributos, valores de atributos e muito mais. É baseado nos seletores CSS existentes e, além disso, possui alguns seletores personalizados próprios. Todos os seletores em jQuery começam com o cifrão e parênteses: $ ().

Seletores de elementos

Se especificar o nome do elemento, então todos os elementos dentro do documento HTML serão selecionados. No exemplo abaixo, iremos selecionar todos os elementos botões, e dado o evento click, iremos selecionar todos os elementos <p> e escondê-los

<script>
  $(document).ready(function () {
    $("button").click(function () {
      $("p").hide();
    });
  });
</script>

Seletores de Identificadores

Se especificar o nome do identificador precedido pelo caractere de hashtag (#), então todos os elementos com aquele Identificador dentro do documento HTML serão selecionados. No exemplo abaixo, iremos selecionar todos os elementos botões, e dado o evento click, iremos selecionar todos os elementos com o identificador de #test e escondê-los

$(document).ready(function(){
  $("button").click(function(){
    $("#test").hide();
  });
});

Seletores de Classes

Se especificar o nome da classe precedido pelo caractere de hashtag (.), então todos os elementos com aquele Identificador dentro do documento HTML serão selecionados. No exemplo abaixo, iremos selecionar todos os elementos botões, e dado o evento click, iremos selecionar todos os elementos com o nome da classe de .test e escondê-los

$(document).ready(function () {
  $("button").click(function () {
    $(".test").hide();
  });
});

Seletor THIS

O seletor this é usado para selecionar o elemento atual/último elemento selecionado.

$("p").on({
  mouseenter: function(){
    $(this).css("background-color", "lightgray");
  },
  mouseleave: function(){
    $(this).css("background-color", "lightblue");
  },
  click: function(){
    $(this).css("background-color", "yellow");
  }
});
