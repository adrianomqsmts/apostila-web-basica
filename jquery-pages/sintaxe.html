Sintaxe

A sintaxe jQuery é feita sob medida para selecionar elementos HTML e executar alguma ação no (s) elemento (s). A sintaxe básica é: $(seletor).ação()

  • Um sinal $ para definir / acessar jQuery
  • Um (seletor) para "consultar (ou localizar)" elementos HTML
  • Uma ação jQuery() a ser realizada no(s) elemento(s)

Exemplos

  • $(this).hide() - oculta o elemento atual.
  • $("p").hide() - oculta todos os elementos <p>.
  • $(".test").hide() - oculta todos os elementos com class = "test".
  • $("#test").hide() - oculta o elemento com id = "teste".
