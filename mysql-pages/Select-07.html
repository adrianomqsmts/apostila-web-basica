Funções do sistema em MySQL

Comas funções do sistema, podemos ver o atual usuário logado no Banco de dados, o banco de dados atual, a versão do sistema SGBD, dentre outras.

-- Vê as regras do usuário
SELECT CURRENT_ROLE();

CURRENT_ROLE();
NONE

-- Vê o nome do usuário atual
SELECT CURRENT_USER();

CURRENT_USER();
root@localhost

-- Vê o banco de dados ativo
SELECT DATABASE();

DATABASE();
apostila

-- Vê a versão do SGBD
SELECT VERSION();

VERSION();
8.0.17

-- Vê o último valor do AUTO_INCREMENTE inserido
SELECT LAST_INSERT_ID();

LAST_INSERT_ID();
0

-- Vê o ID da conexão atual
SELECT CONNECTION_ID();

CONNECTION_ID();
31
