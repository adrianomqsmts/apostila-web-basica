Funções nulas MySQL

Para evitar que ocorram erros no MySQL quando há o retorno de uma coluna com algum dado nulo, precisamos do auxilia das funções nulas para impedir resultados inconsistentes dentro de nossas consultas.

CREATE TABLE produtos(
  -- Chave primária do produto
  id_produto INT primary key auto_increment,
  nome TEXT,
  preco DOUBLE
);

CREATE TABLE pedidos(
  -- Chave primária do pedido
  id_pedido INT primary key,
  id_produto INT,
  quantidade INT,
  -- Criando uma chave estrageira do produto no pedido
  foreign key (id_produto) references produtos (id_produto)
);

IFNULL - Alternativa para valores NULL

A função IFNULL(coluna, valorAlternativo) é usada para substituir o valor da coluna que vir nulo, por um valor alternativo. Uma função alternativa seria COALESCE(), que funciona da mesma forma.

SELECT quantidade, IFNULL(quantidade, 0) FROM pedidos;

quantidade IFNULL(quantidade, 0)
2 2
1 1
NULL 0
3 3
1 1
NULL 0
