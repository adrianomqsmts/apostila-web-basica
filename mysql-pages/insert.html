Inserindo dados com MySQL

Para inserir dados em uma tabela precisamos usar a instrução INSERT INTO. Um detalhe importante é de que dados inseridos do tipo textos devem estar entre aspas simples ' ', enquanto datas devem seguir os padrões já especificados em Tipos de Dados. Mas relembrando 'aaaa-mm-dd' para DATE, 1999-06-24 07:32:00' para DATETIME e '07:32:01' para TIME.

Existem "três" formas distintas de inserir dados em uma tabela, a primeira usamos a instrução completa INSERT INTO nomeTable (nomeColuna1, nomeColuna2, ...) VALUES (dadoColuna1, dadoColuna2,...), onde devemos especificar o nome da tabela e todas as colunas que queremos inserir os dados, seguido pelos dados que serão inseridos. Os dados devem estar na mesma ordem que os nomes das colunas especificadas.

A segunda forma é omitindo o nome das colunas, mas neste caso devemos, nos valores, inserir os dados na mesma ordem das colunas na hora em que a tabela foi criada, e por fim a terceira forma é igual a primeira, mas omitindo algumas colunas e valores que não queremos inserir na tabela.

CREATE TABLE produtos(
  id_produto INT primary key auto_increment,
  nome TEXT,
  preco DOUBLE
);

INSERT INTO produtos (id_produto, nome, preco) values (1, 'Teclado', 10.50);
INSERT INTO produtos (nome, preco) values ('Mouse', 9.90);
INSERT INTO produtos values (3, 'Monitor', 99.90);

Tabela Resultante

id_produto nome preco
1 Teclado 10.5
2 Mouse 9.9
3 Monitor 99.9

Exemplo 2 : Valores Default

Se existem colunas com valores DEFAULT e for de nosso interessem que o valor padrão seja atribuído àquele dado, então basta omití-lo durante a inserção do dado.

CREATE TABLE Conta(
  numero INT primary key auto_increment,
  agencia INT not null,
  saldo FLOAT default 0,
  limite FLOAT default 500.00,
  cartao VARCHAR(3) default 'não'
);

INSERT INTO Conta (agencia) VALUES (987);

Table resultante

numero agencia saldo limite cartao
1 987 0 500 não
