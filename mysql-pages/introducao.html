Introdução ao MySQL

O SQL (Structured Query Language - Linguagem de consulta estruturada) é uma linguagem de acesso e manipulação de banco de dados, como consulta aos dados armazenados, inserção, atualização e exclusão destes dados (CRUD).

Apesar de existir um padrão para o SQL, ainda existem variações entre as empresas quanto a sintaxe, e neste tutorial focaremos nos bancos de dados MySQL. Um banco de dados pode ser visto como uma coleção de uma ou mais tabelas, onde cada dado ou registro é representado por uma linha dentro da tabela.

Nome Idade Salário
Fulano 57 2976,90
Sicrano 23 1676,90
Beltrano 35 2376,90

Diferente de muitas linguagem o SQL não difere letras maiúsculas e letras minúsculas, ou seja, Abc é igual a abc, entretanto existem alguns padrões de boas práticas a serem seguidos. Por fim, é importante ressaltar que cada comando deve-se, por recomendação, terminar com ponto e vírgulas ;, mesmo que não seja obrigatório.

Criando um banco de dados

Após instalar o seu SGBD (Sistema de gerenciamento de banco de dados), abra um arquivo .sql e digite a instrução CREATE DATABASE para criar um banco de dados, seguido pelo nome que deseja dar ao banco.

CREATE DATABASE nomeBanco;

Ou então use a instrução CREATE SCHEMA

CREATE SCHEMA nomeBanco;

Visualizando os bancos de dados

Para visualizar todos os bancos de dados criados use a instrução SHOW DATABASES.

SHOW DATABASES;

Apagando um banco de dados

Para apagar um banco de dados e todos os seus dados, basta usar o comando DROP DATABASE, seguido pelo nome do banco de dados que deseja apagar.

DROP DATABASE nomeBanco;
