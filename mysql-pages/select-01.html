WHERE - Filtrando os dados

A cláusula WHERE é usada em conjunto com a instrução SELECT para filtrar a consulta através de um determinada condição booleana como preço mais que 10 unidades monetárias ou mostrar somente os dados onde a pessoa tem mais de 18 anos.

Os principais operadores lógicos são:

  • WHERE coluna = valor - ds dados sejam iguais ao parâmetro especificado
  • WHERE coluna <> valor - ds dados sejam diferentes ao parâmetro especificado
  • WHERE coluna >= valor - ds dados sejam maiores ou iguais ao parâmetro especificado
  • WHERE coluna <= valor - ds dados sejam menores ou iguais ao parâmetro especificado
  • WHERE coluna > valor - ds dados sejam maiores ao parâmetro especificado
  • WHERE coluna < valor - ds dados sejam menores ao parâmetro especificado

ORDER BY - Ordenando os resultados

A instrução SELECT coluna1, coluna2, .. FROM tabela ORDER BY coluna (DESC) é usada para ordenar as linhas da tabela. Se for especificada a condição DESC os dados serão ordenados de forma decrescente de acordo com a cola especificada, e caso a condição seja omitida, então os dados serão ordenados de forma crescente.

SELECT DISTINCT estado AS 'Estados Cadastrados' FROM Fornecedores ORDER BY estado;

SELECT DISTINCT estado AS 'Estados Cadastrados' FROM Fornecedores ORDER BY estado DESC;

LIKE - Selecionando de acordo com os textos

A instrução SELECT coluna1, coluna2, ... FROM tabela WHERE coluna LIKE "condição" é usada para pesquisar e selecionar colunas com textos que tenha a condição especificada.

  • % - Representa zero, um ou mais caracteres quaisquer
  • _ - Representa um único caractere qualquer
-- Seleciona todas as cidades que começam com b
SELECT * FROM Fornecedores WHERE cidade LIKE 'b%';

-- Seleciona todas as cidades que terminam Com O
SELECT * FROM Fornecedores WHERE cidade LIKE '%O';

-- Seleciona todas as cidades que tem o caractere a
SELECT * FROM Fornecedores where cidade LIKE '%a%';

-- Seleciona todas as razões sociais que tem a segunda letra e
SELECT * FROM Fornecedores WHERE ragsoc LIKE '_e%';

-- Seleciona todas as razões sociais que tem como antepenúltima letra o caractere n
SELECT * FROM Fornecedores WHERE ragsoc LIKE '%n__' ;

-- Seleciona todos os CNPJs que começam com o número 5
SELECT * FROM fornecedores where cnpj LIKE '5%';

LIMIT - Limitando o número de linhas da consulta

O operador LIMIT é usado quando queremos limitar o números de linhas retornadas um uma consulta. Por exemplo, se quisermos que uma consulta retorne apenas uma linhas ao invés de várias, usamos a consulta LIMIT 1, e se quisermos que sejam retornadas até 10 linhas, usamos LIMITE 10. A sintaxe base é SELECT coluna1, coluna2, ... FROM tabela (WHERE condição) LIMITE quantidade.


SELECT * FROM Fornecedores WHERE cidade LIKE '%O'LIMIT 2;
