Selecionando com funções matemáticas MySQL

Existem diversas funções matemáticas prontas no MySQL para manipularmos os dados na hora de selecionarmos em uma tabela resultante. Usaremos os dados abaixo como exemplos.

CREATE TABLE produtos(
  id_produto INT primary key auto_increment,
  nome TEXT,
  preco DOUBLE
);

INSERT INTO produtos (id_produto, nome, preco) values (1, 'Teclado', 10.50);
INSERT INTO produtos (nome, preco) values ('Mouse', 9.90);
INSERT INTO produtos values (3, 'Monitor', 99.90);

SELECT * FROM produtos;

id_produto nome preco
1 Teclado 10.5
2 Mouse 9.9
3 Monitor 99.9

COUNT - Contando os dados

A função SELECT COUNT(coluna1), COUNT(colunA2), ... FROM tabela é usada para contar a quantidade de linhas preenchidas naquela coluna. Pode ser usado para ver quais colunas contém mais dados ausentes.

SELECT count(preco), count(nome) FROM produtos;

count(preco) count(nome)
3 3

SELECT count(*) FROM produtos;

count(*)
3

AVG - Calculando a média

A função SELECT AVG(coluna1), AVG(colunA2), ... FROM tabela é usada para calcular a média dos dados da coluna especificada.

SELECT AVG(preco) FROM produtos;
avg(preco)
40.1

MAX e MIX - Selecionando os valores máximos e mínimos

A função SELECT MAX(coluna1), MIN(coluna2), MIN(coluna3), MAX(coluna4) ... FROM tabela seleciona o valor máximo ou mínimo da coluna especificada.

SELECT MAX(preco), MIN(preco) FROM produtos;

max(preco) min(preco)
99.9 9.9

SUM - Somando os valores de uma coluna

A instrução SELECT SUM(coluna1), SUM(coluna2),... FROM tabela é usada para somar os todos os dados de uma coluna.

SELECT SUM(preco) FROM produtos;

sum(preco)
120.30000000000001

GROUP BY - Agrupando Resultado

HAVING - Cláusula WHERE para o GRUPO BY

A cláusula HAVING funciona como um condição WHERE para o GROUP BY, uma vez que a cláusula WHERE não pode ter as condições de operações, por assim dizer. A sintaxe base é SELECT cont(*), coluna,... FROM tabela GROUP BY coluna HAVING condição;. OBS: Tanto o WHERE quanto o HAVING podem ser usados em conjunto

SELECT estado, count(*) AS 'Quantidade de fornecedores' FROM fornecedores WHERE (ESTADO LIKE 'M%') OR  (ESTADO Like 'S%') GROUP BY estado HAVING count(*) >= 1;

Operadores Matemáticos

Também podemos somar, subtrair, dividir e multiplicar duas colunas em MySQL usando os operadores matemáticos básicos como + - / *.

  a float,
  b float
);

INSERT INTO valores values (1.2, 2.3), (5.5, 8.0), (10.1, 7.4);
