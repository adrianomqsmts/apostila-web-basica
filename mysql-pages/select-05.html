Funções de Datas em MySQL

Existem diversas funções prontas para nos ajudam a trabalhar com datas e horas dentro de um banco de dados, como recuperar a hora atual, dia da semana, o ano, e várias outras possibilidades.

-- Data atual
SELECT current_date();

current_date()
2020-11-26

-- Hora Atual
SELECT current_time();

current_time()
22:29:23

-- Data e hora atual (outras funções now())
SELECT current_timestamp();

current_timestamp()
2020-11-26 22:29:49

CREATE TABLE compras(
  valor DOUBLE,
  dia DATETIME
);

INSERT INTO compras VALUES
(159.99, now()),
(767.99, '2017-11-26 22:40:39'),
(11.90, '1999-05-01 17:30:00');

SELECT * FROM compras;

valor dia
159.99 2020-11-26 22:41:31
767.99 2017-11-26 22:40:39
11.9 1999-05-01 17:30:00

Lista de Funções

Para ver a lista completa com exemplos de funções como extrair o dia da semana, estação do ano, o mês as horas, além de funções que convertem horas em segundos, dias em minutos. Existem funções que calculam a diferença entre duas datas e até funções que retornam a data atual somada a uma outra data. A lista complete de funções pode ser acessada em Date and Time Functions.
