Arrays em PHP

De forma simplista, um array é um vetor que armazena vários valores em uma única variável.

<?php
$lista = ["carro", "moto", "casa"]; //Cria um array de 3 posições
$numeros = array(1,2,3);//Cria um array de 3 posições
?>

Acessando e alterando um valor de um array

Para acessar um valor de um array basta passar um index com a posição do vetor que deseja acessar, onde o index 0 é a primeira posição, o index 1 é a segunda, e assim sucessivamente.

<?php
$lista = ["carro", "casa", "moto"]; //Cria um array de 3 posições
echo $lista[0] . "<br>"; //Exibe o valor carro
$lista[0] = "barco"; //Sobrescreve o valor carro com o valor barco
echo $lista[0]; //Exibe o valor barco
?>

Alterando o index de um array

Podemos mudar o index (normalmente números) de um array para isto, basta usar a sintaxe ["index" => elemento, "index2" => elemento2, ...]. Após alteramos poderemos usar lista["index"] ao invés de lista[0], lembrando que se usarmos agora lista[0], haverá um erro.

<?php
$lista = ["c" => "carro", "m" => "moto", "ca" => "casa"]; //Cria um array de 3 posições
$numeros = array("n1" => 1, "n2" => 2, "n3" => 3); //Cria um array de 3 posições
echo $lista["c"]; //Exibe o valor carro
echo $numeros["n2"]; //Exibe o valor 2
?>

Inserindo novos elementos em um array

Existem diferentes formas de introduzir um novo elemento em um array, a primeira é usando colchetes vazios $lista[] = novoElemento, a segunda é especificando a posição do novo elemento como $lista[ultimaPosicao+1] = novoElemento e a terceira é utilizando a função array_push($lista, novoElemento).

<?php
$lista = ["carro", "casa", "moto"]; //Uma lista de 3 posições
$lista[4] = "bicicleta"; //Insere um elemento na quarta posição
$lista[] = "barco"; //É o mesmo que $lista[5], e insere um elemento na quinta posição
array_push($lista, "skate"); //insere um elemento na sexta posição
echo var_dump($lista);
//Exibe a lista array(6) { [0]=> string(5) "carro" [1]=> string(4) "casa" [2]=> string(4) "moto" [4]=> string(9) "bicicleta" [5]=> string(5) "barco" [6]=> string(5) "skate" }
?>

Array de arrays ou simplesmente matrizes

Arrays multidimensionais são uma matriz, ou um vetor de vetores. Para criar uma matriz, basta inserir como cada elemento de um array, um outro array.

<?php
  $lista = [["arroz", "feijao"], ["carro", "moto"], ["cartão", "dinheiro"]]; //Cria um array de 3 posições
  $numeros = array(array(1, 2, 3), array(5, 6, 7), array(20, 30, 40)); //Cria um array de 3 posições
  echo $lista[0][0] . " " . $numeros[2][2]; //Exibe Arroz 40
?>

Principais funções com array


<?php
$lista = ["carro", "casa", "moto"]; //Uma lista de 3 posições
$lista2 = ["bicicleta", "skate", "barco"]; //Uma lista de 3 posições

$lista3 = array_merge($lista, $lista2); //Retorna um lista com 6 posições

//Retorna uma string do array e a exibe: carro - casa - moto - bicicleta - skate - barco
echo implode(" - ", $lista3) . "<br>";

echo array_pop($lista3) . "<br>"; //Tira o barco da lista e o exibe

array_unshift($lista3, "avião"); //Insere o elemento avião no início da lista3

echo array_shift($lista3); //Remove o elemento avião da lista3 e o exibe

echo array_search("Avião", $lista3); //Pesquisa por avião e retorna nul.
?>


Função : Retorno Descrição
array_intersect(array1, array2, ...) : array Calcula a interseção entre arrays considerando os valores dos elementos (tem variações como array_intersect_key)
array_merge(array1, array2, ...) : array Combina um ou mais arrays (tem variações como array_merge_recursive)
array_key_exists(chave, key): bool Checa se uma chave ou índice existe em um array
array_keys(array, valor): array Retorna todas as chaves ou uma parte das chaves de um array (valor opcional)( tem variações como array_keys_first)
array_push(array, novoElemento1, novoElemento2,...) : int Adiciona um ou mais elementos no final de um array
array_pop(array) : tipoElemento Extrai um elemento do final do array e o retorna
array_unshift(array, novoElemento1, novoElemento2,...) : int Adiciona um ou mais elementos no início de um array
array_shift(array) : tipoElemento Retira o primeiro elemento de um array e o retorna
array_rand(array, quantidade) : array ou tipoElemento Escolhe um ou mais elementos aleatórios de um array e os retorna as chaves
array_search(valor, array) : tipoChave Procura por um valor em um array e retorna sua chave correspondente caso seja encontrado, caso contrário retorna NULL.
array_unique(array) : array Remove os valores duplicados de um array e retorna um novo array
array_values(array) : array Retorna todos os valores de um array
sort(array, tipoOrdenação) : boll Ordena um array, onde o tipo é opcional, podendo ser SORT_REGULAR, SORT_NUMERIC, SORT_STRING, etc
is_array(array) : bool Verifica se a variável é um array
explode("delimitador", string) : array Divide uma string em strings de acordo com o delimitador.
implode("delimitador", array) : string Une todos os elemento de um array em uma string, separados pelo delimitador.
