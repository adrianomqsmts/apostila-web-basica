Banco de dados com PHP

Usaremos o sistema de banco de dados MySQL, que tem como estrutura o armazenamento de dados em tabelas com colunas e linhas. Existem duas principais formas de conectar a páginas PHP a um banco de dados, a primeira é o MySQLi e a segunda forma é o PDO. Ambos possuem vantagem e desvantagem e existem diversas discussões sobre qual a melhor para a sua aplicação e cabe você a decidir isto.

Por tanto, iremos abordar as três formas neste pequeno tutorial. Sim, não escrevi errado, pois o MySQLi pode ser orientado a objetos ou estrutural, então existem "três formas de fazer a mesma coisa".

Conectando ao Banco de dados

O primeira passos quando se trabalha com banco de dados é iniciar a conexão com o banco de dados.

mysqli() - Conectando com MySQLi orientado a objetos

O objeto mysqli($servidor, $usuario, $senha, $nome_banco) é usado para criar um objeto de conexão com o banco de dados.

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "nome_banco";

  // Criando a conexão
  $conn = new mysqli($servername, $username, $password, $dbname);

  // Verificando se ocorreu erro, ou não ao criar a conexão
  if ($conn->connect_error) {
    //Exibindo o que provocou o erro e interrompendo a aplicação
    die("Falha na conexão: " . $conn->connect_error);
  }
  //Exibe que a conexão foi um sucesso
  echo "Conexão estabelecida com sucesso.";
?>

PDO() - Conectando com o PDO

  <?php
    $servername = "localhost";
    $username = "root";
    $password = "";
    $dbname = "nome_banco";
    
    // Iremos tentar fazer a conexão. Em caso de erro, iremos capturá-lo.
    try {
      // Criando a conexão - devemos dizer também o nome do banco de dados que iremos conectar
      $conn = new PDO("mysql:host=$servername;dbname=$dbname", $username, $password);

      $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
      echo "Conexão estabelecida com sucesso.";
    } catch (PDOException $e) {
      //Exibindo o que provocou o erro e interrompendo a aplicação
      echo "Falha na conexão: " . $e->getMessage();
    }
  ?>

Fechando a conexão com o banco de dados

A conexão geralmente é fechada automáticamente após a execução do script, e caso queira fechar antes, ou garantir que a conexão seja fechada para evitar quaisquer problemas futuros, use os comandos abaixo.

  //MySQLi OO
  $conn->close(); 

  //MySQLi Procedural
  mysqli_close($conn);
  
  //PDO
  $conn = null; 

Instruções preparadas

Instruções preparadas (Prepared Statements) é uma forma de tornamos nossas consultas mais seguras, eficientes e modulares, pois nos permitem escrever instruções SQL com parâmetros variáveis e não uma consulta estática.

Assim que uma instrução preparada é criada, ela já é enviada para o banco de dados com certos parâmetros pendentes, e desta forma o próprio banco analisa e otimiza a consulta sem executá-la, esperando apenas os parâmetros restantes para executá-la. Este processo gera um ganho em desempenho, pois a consulta já está otimizada e já no banco de dados, além de reduzir o tamanho da banda, pois somente um parte da consulta é enviada a cada nova instrução SQL.

Por fim, as instruções preparadas são úteis contra SQL injection, pois o modelo SQL já está dentro do banco de dados, assim, caso haja um modelo de instrução diferente do original, a injeção SQL não poderá ocorrer. Para ler um artigo completo sobre SQL Injection e como tratá-lo, acesse PHP MySQLi Prepared Statements Tutorial to Prevent SQL Injection.

Os nossos exemplo abaixo, já estão com instruções preparadas aplicadas.

INSERT - Inserindo dados no Banco de Dados

Para inserir dados bastamos criar uma string contendo o comando SQL e executar o comando. O mesmo processo pode ser usado para criar tabelas e até mesmo um nome para banco de dados, caso preferir. Após a execução precisamos fechar a instrução preparada para evitar quaisquer problemas futuros.

CREATE TABLE testes (
  cpf VARCHAR(14)  PRIMARY KEY,
  nome VARCHAR(100) NOT NULL,
  idade SMALLINT NOT NULL,
  sexo CHAR(1) NOT NULL
);

prepare() e bind_param() - Inserindo dados com MySQLi Orientado a Objetos

O método bind_param() irá ligar as variáveis esperadas ao tipo de dados do primeiro parâmetro da função que pode ser:

  • i - integer
  • d - double
  • s - string
  • b - BLOB

Assim, bond_param("sdi", $var1, $var2, $var3) estará esperando que a primeira variável seja do tipo string, que a segunda seja do tipo double, e que a terceira variável seja do tipo integer, minimizando assim, as chaves de SQL Injection.

<?php
    $servername = "localhost";
    $username = "root";
    $password = "";
    $dbname = "nome_banco";

    // Criando a conexão
    $conn = new mysqli($servername, $username, $password, $dbname);

    // Verificando se ocorreu erro, ou não ao criar a conexão
    if ($conn->connect_error) {
      //Exibindo o que provocou o erro e interrompendo a aplicação
      die("Falha na conexão: " . $conn->connect_error);
    }

    //Cria uma instrução preparada que espera 4 parâmetros (?)
    $stmt = $conn->prepare("INSERT INTO testes (cpf, nome, idade, sexo) VALUES (?, ?, ?, ?)");

    //Ligando as variáveis aos parâmetros esperados
    $stmt->bind_param("ssis",$cpf, $nome, $idade, $sexo);

    // Atribuindo os parâmetros 
    $cpf = "123.123.123-09";
    $nome = "Fulano de Tal";
    $idade = 89;
    $sexo = "M";
    //Executando a Instrução SQL
    $stmt->execute();

    echo "Os dados Foram inseridos com sucesso!";

    //Fechando a instrução preparada
    $stmt->close();
    
    //Fechando a conexão
    $conn->close();
  ?>

get_result() e fetch_assoc() - Selecionado dados com o MySQL orientado a objetos

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "nome_banco";


  // Criando a conexão
  $conn = new mysqli($servername, $username, $password, $dbname);

  // Verificando se ocorreu erro, ou não ao criar a conexão
  if ($conn->connect_error) {
    //Exibindo o que provocou o erro e interrompendo a aplicação
    die("Falha na conexão: " . $conn->connect_error);
  }

  $sql = "SELECT cpf, nome, idade, sexo FROM testes WHERE idade >= ?";

  $stmt = $conn->prepare($sql);
  //Ligando as variáveis aos parâmetros esperados
  $stmt->bind_param("i", $idade);

  $idade = 50;

  // Executando o cógido SQL e armazenando o resultado da consulta no objeto $result
  $stmt->execute();
  $result = $stmt->get_result();

  // $result tem uma propriedade que diz o número de linhas retornada na consulta
  if ($result->num_rows > 0) {
    // Estamos executando o método do fetch_assoc() objeto
    // $result que retorna um array com os dados de uma linha
    // O index do array contém o nome da coluna, em seguida fetch_assoc()
    // aponta para a próxima linha até que row seja FALSE
    while ($row = $result->fetch_assoc()) {
      echo "CPF: " . $row["cpf"] . " - Nome: " . $row["nome"] . ". <br/> Idade - " . $row["idade"] . " Sexo - " . $row["sexo"] . "<br/>" ;
    }
  } else {
    echo "Nenhum resultado foi retornado";
  }
  //Fechando a instrução preparada
  $stmt->close();

  //Fechando a conexão
  $conn->close();
?>

fetchAll() - Selecionando dados com PDO

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "nome_banco";

  // Iremos tentar fazer a conexão. Em caso de erro, iremos capturá-lo.
  try {
    // Criando a conexão - devemos dizer também o nome do banco de dados que iremos conectar
    $conn = new PDO("mysql:host=$servername;dbname=$dbname", $username, $password);

    $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

    $stmt = $conn->prepare("SELECT cpf, nome, idade, sexo FROM testes WHERE idade > :idade");
    $stmt->bindParam(':idade', $idade);
    $idade = 50;
    $stmt->execute();

    // defina a matriz resultante como associativa
    $result = $stmt->fetchAll();

  // O método fetchAll() retorna um array com todos os dados, caso não haja dados retorna FALSE
  if ($result) {
      // Como $result é um array multidimensional podemos iterar entre cada linha
      // usando um foreach
      foreach ($result as $row) {
        echo "CPF: " . $row["cpf"] . " - Nome: " . $row["nome"] . ". <br/> Idade - " . $row["idade"] . " Sexo - " . $row["sexo"] . "<br/>";
      }
    }else{
      echo "Nenhum Resultado Retornado.";
    }

    $stmt = null;
  } catch (PDOException $e) {
    //Exibindo o que provocou o erro e interrompendo a aplicação
    echo "Falha na conexão: " . $e->getMessage();
  }
  //Fechando a conexão
  $conn = null;
?>

Atualizando o banco de dados com MySQLi orientado a objetos

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "nome_banco";


  // Criando a conexão
  $conn = new mysqli($servername, $username, $password, $dbname);

  // Verificando se ocorreu erro, ou não ao criar a conexão
  if ($conn->connect_error) {
    //Exibindo o que provocou o erro e interrompendo a aplicação
    die("Falha na conexão: " . $conn->connect_error);
  }

  $sql = "UPDATE testes SET idade = ? WHERE cpf = ?";

  $stmt = $conn->prepare($sql);
  //Ligando as variáveis aos parâmetros esperados
  $stmt->bind_param("is", $idade, $cpf);

  $idade = 25;
  $cpf = '123.123.123-09';

  // Executando o cógido SQL e armazenando o resultado da consulta no objeto $result
  $stmt->execute();

  // $stmt tem uma propriedade que diz o número de linhas afetadas pela instrução
  if ($stmt->affected_rows > 0) {
    echo "Dado(s) Atualizados(s) com sucesso";
  } else {
    echo "Nenhum dado foi atualizado";
  }
  //Fechando a instrução preparada
  $stmt->close();

  //Fechando a conexão
  $conn->close();
?>

Atualizando dados com PDO

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "nome_banco";

  // Iremos tentar fazer a conexão. Em caso de erro, iremos capturá-lo.
  try {
    // Criando a conexão - devemos dizer também o nome do banco de dados que iremos conectar
    $conn = new PDO("mysql:host=$servername;dbname=$dbname", $username, $password);

    $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);

    $stmt = $conn->prepare("UPDATE testes SET idade = :idade WHERE cpf = :cpf");
    $stmt->bindParam(':idade', $idade);
    $stmt->bindParam(':cpf', $cpf);
    $idade = 33;
    $cpf = '123.123.123-09';
    $stmt->execute();

    // O método rowCount() retorna quantas linhas foram afetadas durante 
    // o uso da conexão
    if ($stmt->rowCount() > 0) {
      echo "Dado(s) atualizado(s) com sucesso";
    } else {
      echo "Nenhum dado foi atualizado";
    }


    $stmt = null;
  } catch (PDOException $e) {
    //Exibindo o que provocou o erro e interrompendo a aplicação
    echo "Falha na conexão: " . $e->getMessage();
  }
  //Fechando a conexão
  $conn = null;
?>
