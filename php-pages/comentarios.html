Comentários em PHP

Neste ponto, já deve saber a importância de se usar comentários... Mas sempre é bom relembrar que comentários são úteis para ajudar a outros programadores, incluindo você, a entender o código e as decisões tomadas. Além claro de ajudar a encontrar erros, pois podemos comentar partes do código para que a parte comentada não seja executada, impedindo de ter que apagar e reescrever esses trechos a cada teste.

<?php
// Este é um comentário de linha única

# Este é outro comentário de linha única

/*

Este é um comentário de 
múltiplas linhas

*/ 
?>
