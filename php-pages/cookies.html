Cookies com PHP

Cookies são pequenos arquivos que o servidor trabalha no computador do usuário. Assim, sempre o um computador solicitar um página, ele envia junto o cookie, permitindo o servidor trabalhar com estes dados, reconhecendo o usuário, por exemplo.

setcookie() - Criando cookies

A função setcookie(nome, valor, validade, caminho, domínio, segurança, httpApenas) é usada para criar um cookie, onde somente o valor nome é obrigatório, enquanto os demais parâmetros são opcionais. A função setcookie() aparecer ANTES da tag <html>

<?php
  $nome = "usuario";
  $valor = "Fulano de Tal";
  //time retorna a hora atual e somaremos mais 7 dias até o cookie expirar
  $validade = time() + (7 * 24 * 60 * 60);
  setcookie($nome, $valor, $validade, "/");
?>

Caso queira saber mais sobre eles acesse setcookie.

Variável super global $_COOKIE

A variável super global $_COOKIE[$nome] contém o valor do nome especificado dentro de um cookie. Normalmente usamos a função isset() para ver se o valor do cookie existe, antes de trabalharmos com ele.

<?php
  if (!isset($_COOKIE["usuario"])) {
    echo "Nenhum valor de usuário definido!";
  } else {
    echo "Bem vindo " . $_COOKIE["usuario"];
  }
?>

Alterando o valor de um cookie

Para alterar o valor de um cookie, basta usar novamente a função setcookie(nome, valor, validade, caminho, domínio, segurança, httpApenas), com o mesmo nome do valor que deseja alterar

<?php
  $nome = "usuario";
  $valor = "Sicrano de Tal";
  //time retorna a hora atual e somaremos mais 7 dias até o cookie expirar
  $validade = time() + (7 * 24 * 60 * 60);
  setcookie($nome, $valor, $validade, "/");
?>

Apagando um cookie

Para deletar um cookie, basta atribuir uma data de validade passada! Assim o cookie irá expirar e apagar.

<?php
  //time retorna a hora atual e subtrairemos 7 dias para o cookie expirar
  $validade = time() - (7 * 24 * 60 * 60);
  setcookie("","", $validade, "/");
?>
