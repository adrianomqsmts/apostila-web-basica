Escopos em PHP

De forma resumida um escopo define onde uma variável pode ver vista/acessada. Em PHP um variável pode ser global, local ou estática.

Basicamente, uma variável definida >fora de uma função tem um >escopo global, enquanto uma variável declarada >dentro de uma função tem um >escopo local e somente poderá ser acessada dentro de uma função! As duas formas abaixo estão erradas e podem gerar erros!

 <?php
  $x = 5; // Escopo Global

  function funcGlobal()  {
    //Usar a variável X dentro de uma função vai gerar um erro.
    echo "<p>O valor de X fora da função é: $x</p>";
  }
  funcGlobal();

  echo "<p>O valor da variável x fora da função é: $x</p>";

  function funcLocal()  {
    $y = 5; // Escopo Local
    echo "<p>O valor de y dentro da função é: $y</p>";
  }
  funcLocal();

  // Usar a variável Y fora da função vai gerar erro.
  echo "<p>O valor da variável y fora da função é: $y</p>";
  ?> 

Palavra reservada Global

Para usar uma variável global dentro de uma função precisamos usar a palavra reservada global, que é usada para acessar uma variável global de dentro de uma função.

<?php
$x = 5;
$y = 10;

 function soma() {
   global $x, $y;
   $y = $x + $y;
 }

 soma();
 echo $y; // saida é 15
 ?>

PHP também armazena todas as variáveis globais em um array chamado $GLOBALS[index]. O index contém o nome da variável, e poderemos então, através do index acessar o valor armazenado de uma variável.

 <?php
$x = 5;
$y = 10;

function soma() {
  $GLOBALS['y'] = $GLOBALS['x'] + $GLOBALS['y'];
}

soma();
echo $y; // saída 15
?> 

Palavra reservada static

Normalmente, quando uma função é concluída / executada, todas as suas variáveis locais são excluídas. No entanto, às vezes queremos que uma >variável local NÃO seja excluída. Para evitar esta exclusão precisamos usar a palavra reservada static. Desta forma, quando a função for chamada novamente, o valor da variável local ainda estará armazenado.

 <?php
  function contador()  {
    static $x = 0;
    echo $x++;
  }

  contador();  //Escreve 0
  contador(); //Escreve 1
  contador(); //Escreve 2
  ?>
