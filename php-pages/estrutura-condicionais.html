Estruturas condicionais em PHP

Estruturas condicionais são usadas quando queremos que diferentes ações ocorram em determinados contextos que apresentam diferentes condições.

Estrutura condicional if...elseif...else

  • if - Usado para executar um determinado trecho de código com base em uma condição se for verdadeira.
  • elseif - Usado para executar um determinado trecho de código se a condição if ou if else imediatamente acima não for verdadeira e a condição atual for verdadeira.
  • else- Usado para executar um determinado trecho de código se todas as condições acima não forem verdadeira.
<?php
  $hora = date("H");
  echo "A hora agora é $hora <br>";

  if ($hora < "12" && $hora > "6") {
    echo "Bom dia!";
  } elseif ($hora >= "12" && $hora < "18") {
    echo "Boa tarde!";
  } else {
    echo "Boa noite!";
  }
?>

Estrutura condicional switch

O switch é um estrutura condicional de múltiplos casos, bem semelhante a estrutura IF-ELSE. A estrutura recebe como entrada uma única variável e realiza a comparação de igualdade == ou seja 10 == "10" é verdade.

  • switch(variável) - O switch precisa de um parâmetro único que será usado para realizar a comparação de mesmo valor e tipo em cada caso;
  • case valor: - Cada caso deve ter um valor que será comparado com a variável especificada no switch.
  • break; - É usado para interromper a estrutura, ou seja, quando o break é executado, as próximas linhas do código não serão executadas, caso omitido, todos os casos abaixo serão executados, mesmo que sejam falsos. .
  • default - Caso especial, colocado como último caso, em que sempre será executado quando nenhum outro caso for executado.
<?php
  $cor = "verde";
  switch ($cor) {
    case "vermelho":
      echo "A cor escolhida é vermelha!";
      break;
    case "azul":
      echo "A cor escolhida é azul!";
      break;
    case "verde":
      echo "A cor escolhida é verde!";
      break;
    default:
      echo "A cor escolhida não é vermelha, azul ou verde!";
  }
?>
