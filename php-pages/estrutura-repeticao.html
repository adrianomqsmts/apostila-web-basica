Estruturas de Repetição em PHP

Um estrutura de repetição basicamente é usada para repetir um mesmo trecho de código um número finito de vezes, tendo como objetivo reduzir o número de linhas escritas com o mesmo trecho de código.

SEMPRE ESPECIFIQUE A CONDIÇÃO DE PARADA!!! Caso esqueça ou erre a lógica das condições o laço de repetição nunca irá parar e poderá travar o sistema.

Estrutura de repetição FOR

O for(condição1; condição2; condição3) é usado quando queremos repetir uma QUANTIDADE CONHECIDA de vezes um determinado trecho de código.

  • Condição 1 - É executada uma única vez, geralmente é usada para inicializar o valor de um contador com o valor 0; CONHECIDA COMO CONDIÇÃO INICIAL
  • Condição 2 - É uma condição que sempre será executada no inicio de cada interação e normalmente define um valor limite de quantas vezes queremos que a estrutura seja executada. CONHECIDA COMO CONDIÇÃO DE PARADA
  • Condição 3 - É executada sempre ao final da execução do bloco de código, geralmente é usada para incrementar o valor do contador; CONHECIDA COMO CONDIÇÃO DE INCRIMENTO

As condições podem ser omitidas e a estrutura pode assumir diversos formatos que normalmente podem levar a uma má legibilidade dos códigos e não são muito tradicionais, e por estes motivos não iremos abordar essas possibilidades. Um caso mais comum é usar um segunda condição de parada dentro do código com um if e um break, o mesmo vale para a terceira condição.

<?php
  for ($x = 0; $x <= 100; $x += 10) {
    echo "O número é: $x <br>";
  }
?>

Estrutura de Repetição FOREACH

A estrutura de repetição foreache funciona apenas em arrays e é usado para percorrer cada par de chave/valor em um array. Para cada iteração de repetição, o valor do o elemento do array atual é atribuído a variável especificada e o ponteiro do array é movido até atingir o último elemento da matriz. Existem funções para trabalhar com os ponteiros de um array (prev, next, pos, reset) para saber sobre acesse Ref ponteiros array.

<?php
  $cores = array("Vermelho", "Verde", "Azul", "Amarelo");

  foreach ($cores as $elemento) {
    echo "$elemento <br>";
  }
  /*Saida Vermelho <br> Verde <br> Azul <br> Amarelo*/
?>

Para percorremos as chaves de um array, podemos alterar um pouco a sintaxe apontado para $index => $elemento.

<?php
  $Cores = array("l2" => "Vermelho", "l1" =>  "Verde", "l3" => "Azul", "aaa" =>  "Amarelo");

  foreach ($Cores as $chave => $elemento ) {
    echo "$chave = $elemento <br>";
  }
  /*Saida l2 = Vermelho <br> l1 = Verde <br> l3 = Azul <br> aaa = Amarelo*/
?>

Estrutura de Repetição WHILE

A estrutura de repetição while(condição){ CÓDIGO } é semelhante à estrutura for sem a condição 1 e condição 3. Ou seja, devemos em muitos casos iniciar um valor anteriormente e criar uma condição de incremento dentro do corpo da estrutura. Os principais usos de uma estrutura while estão relacionados quando não sabemos quantas vezes iremos executar o bloco de código. Seu uso mais comum contém a palavra reservada break.

<?php
  $x = 0;

  while ($x <= 100) {
    echo "O número é: $x <br>";
    $x += 10;
  }
?>

Estrutura de repetição DO WHILE

A estrutura de repetição do{ }while(condição) é semelhante à estrutura while. A única diferença que a condição é verificada ao final da execução do código e não no início. Ou seja, o do while executará sempre um única vez independentemente se a condição for verdadeira ou não. Enquanto o while sempre fará a verificação da condição antes de executar o código.

<?php
  $x = 6;

  do {
    echo "O número é: $x <br>";
    $x++;
  } while ($x <= 5);

  //Exibe apenas : O número é 6
?>

Continue VS Break

O break interrompe todo o fluxo da estrutura saindo dela, enquanto o continue apenas interrompe o fluxo da iteração atual, mas não saí do fluxo da estrutura.
