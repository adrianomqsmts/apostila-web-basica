Funções em PHP

Uma função é um trecho de comandos em bloco que faz uma determinada tarefa que pode ser realizada em diferentes pontos de um código e quantas vezes quisermos. As vantagens de se utilizar funções é a clareza do código, pois podemos separar o código em diversas partes facilitando a manutenção do mesmo e a possibilidade de reutilização mais simples (sem ter copiar e colar varias linhas de códigos).

A sintaxe básica de uma função é: function nome(parametro1, paramentro2,...) { código; }. Onde function é a palavra chave para criar uma função com o nome especificado seguido sempre de parenteses que podem ou não conter parâmetros. Os parâmetros são variáveis que irão receber um valor e serem usada dentro do bloco de código da função.

Caso haja um retorno na função podemos especificar o tipo do retorno em uma função usando : tipo. E para definir valores padrão para uma variável, caso a mesma seja ausente, padrão precisamos usar apenas (parâmetro = valor, parâmetro2, ...).

<?php 
  function sum(int $x = 10, int $y = 20) : int{
    $z = $x + $y;
    return $z;
  }

  echo "5 + 10 = " . sum(5, 10) . "<br>"; // retorna 15
  echo "7 + 13 = " . sum(7, 13) . "<br>"; // retorna 20
  echo "soma = " . sum(); //retorna 30
?>

Não precismos especificar o tipo do parâmetro, nem o retorno de uma função, pois em PHP já existe a conversão do tipo. Além disto, em PHP os valores dos parâmetros são por valor, ou seja, uma cópia do valor original, e para serem por referência, ou seja, o valor original será alterado, usamos antes do $ o caractere &.

<?php 
  function sum($x, $y){
    $z = $x + $y;
    return $z;
  }

  echo "5 + 10 = " . sum(5, 10) . "<br>"; // retorna 15
  echo "7 + 13 = " . sum(7, 13) . "<br>"; // retorna 20
  //echo "soma = " . sum(); -> erro, precisa de parâmetros
?>
