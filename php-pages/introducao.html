Introdução ao PHP

PHP é um acrônimo para "PHP: Hypertext Preprocessor", cuja a linguagem é de código aberto, e seus scripts são executados no lado do servidor. Um arquivo com o código-fonte do PHP deve ter a extensão ".php" e podem conter texto, HTML, CSS, JavaScript e, claro, código PHP, onde o resultado retornado ao navegador é um simples HTML.

Como em muitas outras linguagens, o PHP é case sensitive. Além disso, a linguagem é fracamente tipada, ou seja, não é necessário declarar o tipo da variável e sua compilação é interpretada.

Sintaxe Básica

Todos os códigos html deve estar entre o delimitador de abertura <?php e o delimitador de fechamento ?>

<?php ?>
