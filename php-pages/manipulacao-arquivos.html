Manipulação de arquivos com PHP

Muitas vezes podemos querer abrir aquivos para visualizar seus conteúdos e até mesmo editá-los para diversas finalidades, como simular um banco de dados, onde os arquivos são guardados em arquivos, ou um aplicação robusta em que trabalha sobre edição de arquivos.

Modos de leitura e escrita

As vezes queremos apenas abrir um arquivos sem editá-lo, ou abrir e editar todo o seu conteúdo, e para isso precismos especificar o modo como iremos abrir o arquivo, especificando alguns dos seguintes parâmetros.

  • r - Abre um arquivo apenas para leitura, onde o ponteiro do arquivo é posicionado no ínicio.
  • w - Abre um arquivo apenas para a escrita, onde o ponteiro do arquivo é posicionado no ínicio, ou seja, todo o conteúdo escrito, será escrito sobre o conteúdo do arquivo original, e caso o arquivo não exita, é criado um novo arquivo.
  • a - Abre um arquivo apenas para a escrita, onde o ponteiro do arquivo é posicionado no final, ou seja, todo o conteúdo escrito é posicionado no final do arquivo, preservando o conteúdo original. caso o arquivo não exita, é criado um novo arquivo.
  • x - Cria um novo arquivo feito somente para a escrita, retornando um erro caso o arquivo já exista.
  • r+ - Abre um arquivo apenas para leitura e a escrita, onde o ponteiro do arquivo é posicionado no ínicio.
  • w+ - Abre um arquivo apenas para leitura e a escrita, onde o ponteiro do arquivo é posicionado no ínicio, ou seja, todo o conteúdo escrito, será escrito sobre o conteúdo do arquivo original, e caso o arquivo não exita, é criado um novo arquivo.
  • a+ - Abre um arquivo apenas para leitura e a escrita, onde o ponteiro do arquivo é posicionado no final, ou seja, todo o conteúdo escrito é posicionado no final do arquivo, preservando o conteúdo original. caso o arquivo não exita, é criado um novo arquivo.
  • x+ - Cria um novo arquivo feito somente para a leitura e a escrita, retornando um erro caso o arquivo já exista.

fopen() - Abrindo um arquivo

O método fopen("caminho/arquivo.extensão", "modoLeitura") recebe dois parâmetros, o primeiro parâmetro do método contém o nome do arquivo, e a sua extensão, e o segundo parâmetro especifica em qual modo o arquivo deve ser aberto.

Muitas vezes podemos querer usar um método para tratar algum erro na abertura de um arquivo, para isto usamos o método die("mensagem") para tratar o erro na abertura de um arquivo, seja por não existir ou por já existir. Por fim, é recomendado assim que abrir o arquivo, já inserir o método fclose($arquivo) ao final do código, que é usada para fechar o arquivo de maneira correta, impedindo que hajam erros ou inconsistência dos dados.

<?php
  $arquivo = fopen("arquivo.txt", "r") or die("Não foi possível abrir o arquivo");
  fclose($arquivo);
?>

fread() - Lendo o conteúdo de um arquivo

A função fread($arquivo, tamanho)lê a partir de um arquivo aberto. O primeiro parâmetro de da função contém o a variável do arquivo aberto a ser lido e o segundo parâmetro especifica o número máximo de bytes a serem lidos, para isto usamos a função filesize("caminho/arquivo.extensão") que retorna o tamanho em bytes de um arquivo.

<?php
  $arquivo = fopen("arquivo.txt", "r") or die("Não foi possível abrir o arquivo");
  echo fread($arquivo, filesize("arquivo.txt"));
  fclose($arquivo);
?>

fgets() - Lendo linhas de um arquivo

A função fgets($arquivo) é usada para ler uma única linha de um arquivo. Após uma chamada da função, o ponteiro do arquivo será movido para a próxima linha, e assim sucessivamente. Como auxílio usamos a função feof($arquivo) para verificar se o "fim do arquivo" (EOF) foi atingido.

<?php
  $arquivo = fopen("arquivo.txt", "r") or die("Não foi possível abrir o arquivo");
  // Exibe linha por linha do arquivo, até o final do arquivo EOF
  while (!feof($arquivo)) {
    echo fgets($arquivo) . "<br>";
  }
  fclose($arquivo);
?>

fgetc() - Lendo caracteres de um arquivo

A função fgetc($arquivo) é usada para ler um único caractere de um arquivo. Após uma chamada da função, o ponteiro do arquivo será movido para o próximo caractere, e assim sucessivamente. Como auxílio usamos a função feof($arquivo) para verificar se o "fim do arquivo" (EOF) foi atingido.

<?php
  $arquivo = fopen("arquivo.txt", "r") or die("Não foi possível abrir o arquivo");
  // Exibe caractere por caractere (incluindo espaços e quebra de linhas) do arquivo, até o final do arquivo EOF
  while (!feof($arquivo)) {
    echo fgetc($arquivo) . " - ";
  }
  fclose($arquivo);
?>

fwrite() - Escrevendo em um arquivo

A função fwrite($arquivo, "texto") é usada para escrever um conteúdo dentro de um arquivo existente ou novo. Cuidado, que dependendo do modo como foi aberto o arquivo, o conteúdo original pode ser sobreposto ou simplesmente APAGADO.

<?php
  $arquivo = fopen("arquivo.txt", "a+") or die("Não foi possível abrir o arquivo");
  $txt = "FULANO de tal \n";
  fwrite($arquivo, $txt);

  // Como o ponteiro do arquivo foi aberto ao final,
  // precismos voltar o ponteiro para o inicio para exibir o arquivo
  rewind($arquivo);

  while (!feof($arquivo)) {
    echo fgets($arquivo) . " <br> ";
  }
  fclose($arquivo);
?>

Principais funções para a manipulação de arquivos

Função : retorno Descrição
file_exists("caminho/arquivo.extensão") : bool Verifica se um arquivo ou diretório existe
rewind($arquivo) : bool Reinicializa a posição do ponteiro de arquivos para o início
copy($arquivo, $arquivoCopia) : bool Copia o conteúdo de um arquivo em outro arquivo
unlink($arquivo) : bool Apaga um arquivo (desde que não esteja em uso)
disk_free_space("caminho") or diskfreespace("caminho") Retorna o espaço disponível no sistema de arquivos ou partição de disco
fgetss($arquivo) Lê uma linha de um ponteiro de arquivo e retira as tags HTML
file_get_contents($arquivo) : string Lê todo o conteúdo de um arquivo para uma string
fileatime("caminho/arquivo.extensão") : int Obtém o último horário de acesso do arquivo
filesize("caminho/arquivo.extensão") : int Obtém o tamanho do arquivo
readfile("caminho/arquivo.extensão") : int Lê e exibe o conteúdo de um arquivo

Existem outros métodos, como criação e exclusão de diretórios, alteração de permissão de arquivos, mover arquivos entre diretórios, dentre outras funções. Para saber mais acesse Funções para Sistema de arquivo.

Uma atenção especial para as funções sprintf, sscanf e fscanf que são também, muito usados para ler e escrever em arquivos, onde trabalhamos com tipos de dados.
