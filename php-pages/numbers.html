Números em PHP

Um >tipo de dados inteiro (integer) é um número não decimal entre -2147483648 e 2147483647. Um valor maior (ou menor) do que este, será armazenado como flutuante, porque excede o limite de um número inteiro.

O >tipo de dados flutuante (float) pode geralmente armazenar um valor de até 1,7976931348623E + 308 (dependente da plataforma) e têm uma precisão máxima de 14 dígitos.

Um valor numérico maior que o valor de um dado flutuante máximo é considerado infinito, enquanto NaN significa Não é um número. NaN é usado para operações matemáticas impossíveis. Para saber qual o tipo do número devemos usar as funções abaixo.

  • is_int(), is_integer(), is_integer() para saber se é um dado inteiro
  • is_float(), is_double() para saber se um dado é flutuante
  • is_finite(), is_infinite(), var_dump() verifica se um número é infinito
  • is_nan(), var_dump() verifica se o tipo de dado é NaN
  • is_numeric() Descobre se o dado é numérico ou não.
  <?php
  $x = 5985;
  var_dump(is_numeric($x)); // Retorna bool(true)

  echo "<br>";

  $x = "5985";
  var_dump(is_numeric($x)); // Retorna bool(true)

  echo "<br>";

  $x = "59.85" + 100;
  var_dump(is_numeric($x)); // Retorna bool(true)

  echo "<br>";

  $x = "Hello";
  var_dump(is_numeric($x)); // Retorna bool(false)
  ?>

Funções Matemáticas

 <?php

  echo (pi() . "<br/>"); // retorna 3.1415926535898
  $lista = [0, 150, 30, 20, -8, -200];
  echo (min($lista) . "<br/>"); //Retorna -200
  echo (max($lista) . "<br/>"); //Retorna 150
  echo (abs(-6) . "<br/>");  // retorna 6
  echo (sqrt(64) . "<br/>");  // retorna 8
  echo (round(0.60) . "<br/>");  // retorna 1
  echo (round(0.49) . "<br/>");  // retorna 0
  echo (ceil(0.1) . "<br/>");  // retorna 1
  echo (floor(0.9) . "<br/>");  // retorna 0

  ?>
Função Descrição
pi() Retorna o valor de PI
min(0, 150, 30, 20, -8, -200)
min(array)		 Retorna o menor valor dentre os valores especificados
max(0, 150, 30, 20, -8, -200)
max(array)		 Retorna o maior valor dentre os valores especificados
abs(number) Retorna o valor absoluto (positivo) de um número
sqrt(number) Retorna a raiz quadrada de um número
round(number) Arredonda um número de ponto flutuante para o número inteiro mais próximo
ceil(number) Arredonda um número para cima para o inteiro mais próximo
floor(number) Arredonda um número para baixo para o inteiro mais próximo
rand()
rand(min, max)		 Gera um número aleatório (com ou sem um intervalo especificado).
exp(number) Calcula o expoente de e
pow(x,y) Retorna x elevado à potência de y
