Sessões com PHP

Uma sessão é uma maneira de armazenar informações (em variáveis) a serem usadas em várias páginas. Ao contrário de um cookie, as informações não são armazenadas no computador do usuário e por padrão, as variáveis ​​de sessão duram até que o usuário feche o navegador.

A maioria das sessões define uma chave de usuário para o navegador do usuário. Isto permite que, quando a sessão é aberta em outra página, ele verifique a chave e caso haja uma correspondência, a sessão é atribuída ao navegador, se não, ele inicia uma nova sessão.

session_start() - Iniciar uma sessão

Uma sessão é iniciada com a função session_start() e deve ser a primeira coisa em seu documento, antes de qualquer tag HTML!. E para manipular valores de sessões devemos usar a variável super global $_SESSION, que funciona como um array que irá armazenar em um vetor os valores da sessão.

 <?php
    // Iniciando a sessão
    session_start();
  ?>
 <!DOCTYPE html>
 <html>
 <body>
   <?php
      // Criando variáveis de sessão
      $_SESSION["nome"] = "Fulano de Tal";
      $_SESSION["idade"] = 17;
    ?>
 </body>
 </html>

SEMPRE que for trabalhar com sessão deve usar a função session_start() no inicio do documento! SEMPRE!

session_unset() - limpando os valores

Podemos usar a função session_unset() para limpar todos os valores armazenados dentro da variável super global $_SESSION, mas ainda continuar com a sessão aberta.

 <?php
    session_start();
  ?>
 <!DOCTYPE html>
 <html>
 <body>
   <?php
      $_SESSION["nome"] = "Fulano de Tal";
      $_SESSION["idade"] = 17;
      //Limpando dos valores das variáveis
      session_unset();
    ?>
 </body>
 </html>

session_destroy() - interromper a sessão

Podemos usar a função session_destroy() para interromper uma sessão, destruindo todas as variáveis e os índices da variável super global $_SESSION.

 <?php
    session_start();
  ?>
 <!DOCTYPE html>
 <html>
 <body>
   <?php
      $_SESSION["nome"] = "Fulano de Tal";
      $_SESSION["idade"] = 17;
      //Interrompe a sessão e apaga tudo
      session_destroy();
    ?>
 </body>
 </html>
