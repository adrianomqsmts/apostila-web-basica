Strings em PHP

Uma String pode ser vista como um vetor de caracteres (imagem abaixo). Onde o primeiro caractere do vetor é a posição 0, o segundo a posição 1 e assim sucessivamente.


Existem diversas funções que podem ser usadas para manipular strings, abaixo temos as principais destacadas com uma breve descrição.

  <?php

  $texto = "Um pequeno texto";
  echo strlen($texto) . "<br>"; //Retorna 16
  echo strpos($texto, "pequeno") . "<br>"; //Retorna 3 
  echo strcmp("oi", "Oi") . "<br>"; //Retorna false = 1
  echo explode(" ", $texto) . "<br>"; //Retorna ["um", "pequeno", "texto"]
  echo strtoupper("Abacaxi"); //Retorna ABACAXI
  parse_str("nome=Fulano");
  echo $nome; // Escreve Fulano, mesmo sem criar a variável $nome

  ?>
Função Descrição
strlen(string) Retorna o comprimento de uma string
str_word_count(string) Retorna o número de palavras em uma string
strpos(string, palavraPesquisada) Retorna a posição da primeira correspondência da palavra pesquisada, se não encontrar retorna false
str_replace(nova, antiga, string) Substitui a palavra ou cadeia de caracteres antiga, pela nova palavra ou cadeia de caracteres em uma string
implode(separador, array) Retorna uma string com as palavras separadas pelo separador. É o mesmo que a função join(separador , string)
explode(separador, string) Retorna um array com as palavras separadas pelo separador de uma string. É o mesmo que a função join(separador , string)
parse_str("nome=Fulano&idade=47") Analisa uma string de consulta em variáveis. Basicamente criará uma variável $nome = Fulano e $idade = 47
strcmp(string1, string2) Retorna se ambas as string contém o mesmo valor, diferenciando maiúsculas de minúsculas. É o mesmo que a função strncmp(string, string, quantidadeLetrasParaComprar), entretanto ela limita o número de caracteres a serem comprador, e não a string completa
strcasecmp(string, string) Retorna se ambas as string contém o mesmo valor, mas não diferenciando maiúsculas de minúsculas
substr(string, inicio, final) Retorna uma parte de uma string que é delimitada pelos valores especificados.
lcfirst(string) Converte o primeiro caractere de uma string em minúsculas
ucwords(string) Converte o primeiro caractere de cada palavra em uma string em maiúsculas
strtoupper(string) converte uma string em letras maiúsculas
strtolower(string) converte uma string em letras minúsculas
