Variáveis em PHP

Variáveis ​​são "contêineres" para armazenar informações. No PHP, uma variável começa com o caractere especial de cifrão $, seguido pelo nome da variável que desejamos dar para a variável. O nome deve seguir as seguintes regras:

  • Uma variável começa com o caractere especial de cifrão $
  • Um nome de variável deve começar com uma letra ou com um underline _. Um nome de variável não pode começar com um número, mas pode conter caracteres alfanuméricos no seu corpo (A-z, 0-9 e _)
  • Os nomes das variáveis diferenciam maiúsculas de minúsculas ($abc e $Abc são duas variáveis diferentes)

Tipos de variáveis

<?php

$texto = "texto"; //String
$inteiro = 10; //Integer
$flutuante = 15.753; //Float
$booleano = false; // Boolean
$lista = ["carro", "moto", "bicicleta"]; //Array pode ser $lista = array("c","m","b");
$objeto = new string("objeto de string"); //Object
$nulo = null; //Null
$naoNumero = NAN; //NaN não é um número

?>

Podemos ver o tipo de dado de um variável com a ajuda da função var_dump(variável). Esta função irá retorna o tipo da variável.

Constantes

Constantes são como variáveis que não podem ter seus valores alterados e são sempre globais. A função que cria uma contante é define() que possui três parâmetros, o primeiro, o nome da constante, o segundo, o valor, e um terceiro (opcional) especifica se o nome da constante deve ser indiferente a maiúsculas. O padrão é falso. Por convenção nomes de constantes são sempre em letra maiúsculas.

O valor também pode ser uma lista do tipo array.

<?php
  define("CONSTANTE",  782624516);
  echo CONSTANTE; // exibe 782624516
?>

isset() - Verificando se existe uma variável

A função isset($variável) verifica se uma variável existe ou não. Retornado true se existe, e false caso contrário. É muito usada para ver se dados dos formulários foram enviados, se um array, dado o index, está definido, ou se existe um determinado cookie atribuído.

<?php
if (isset($var)) {
    echo "Essa variável existe.";
}
?>

Para destruir uma variável, usamos a função unset($variável)

empty() - Verificando se a variável está vazia

A função empty($variável) verifica se uma variável está vazia. Uma variável é considerada vazia se não existir ou seu valor é igual false. A função empty() não gera um aviso se a variável não existir. Retorna false se a variável existir e não estiver vazia e não conter um valor zerado. Caso contrário retornará true. Exemplos "", 0, 0.0, NULL, false, array(), [] e $var (declarada, mas sem valor).

<?php
if (empty($var)) {
    echo "Essa variável está vazia, contém valor 0 ou não está atribuída um valor.";
}
?>

is_* verificando o tipo da variável

Através das funções abaixo, poderemos saber qual o tipo da variável. Retornam true se o tipo for o da função, e false caso contrário.

  • is_bool($variável) - Verifica se a variável é um boleano
  • is_numeric($variável) - Informa se a variável é um número ou uma string numérica
  • is_float($variável) - Informa se a variável é do tipo float
  • is_int($variável) - Informa se a variável é do tipo inteiro
  • is_string($variável) - Informa se a variável é do tipo string
  • is_object($variável) - Informa se a variável é um objeto
  • is_array($variável) - Verifica se a variável é um array

vardump() - Informações da variável

A função vardump($variável) mostra informações sobre a variável, mostrando toda a estrutura da variável, mostrando também o seu tipo e o seu valor.

<?php
  $array = [1, 2, 3];
  echo var_dump($array);
  // retorna array(3) { [0]=> int(1) [1]=> int(2) [2]=> int(3) }
  // mostrando o tamanho do array, o index de cada elemento e o seu respectivo valor e tipo
  // como $array[2] = 3
?>
